# 浙江公安主题填报系统 — 产品需求文档（PRD）

> 版本：v1.0
> 日期：2026-04-07
> 产品：浙江公安主题填报移动端系统
> 甲方：浙江省公安厅融媒体中心

---

## 1. 产品概述

### 1.1 背景

浙江省公安厅融媒体中心需要一个移动端主题填报系统，用于管理公安主题宣传的稿件收集、主题配置和专题推荐。系统需体现浙江公安特色，兼顾政务严肃性和视觉美观度。

### 1.2 产品定位

面向浙江省公安厅内部工作人员的移动端内容管理工具，核心功能是主题创建、稿件填报和专题推荐配置。

### 1.3 设计风格

**政务蓝 + 水墨浙江**
- 主色调：深蓝渐变（`#0D2B6B` → `#1E4FCC`），体现公安专业与权威
- 强调色：金色（`#C9A04E`），呼应警徽元素
- 装饰元素：水墨山水 SVG 剪影，体现浙江地域特色
- 蓝色竖线装饰条，致敬公安肩章

---

## 2. 系统架构

### 2.1 功能模块

系统包含三个平行模块，本期实现「主题填报」模块，其余两个模块预留入口：

| 模块 | 状态 | 说明 |
|------|------|------|
| **主题填报** | 本期实现 | 主题创建、管理、稿件填报、专题推荐配置 |
| 数据报告 | 预留入口 | 未来实现，用于数据统计与分析 |
| 异常处置 | 预留入口 | 未来实现，用于异常稿件处理 |

### 2.2 页面清单

| 页面 | 文件 | 说明 |
|------|------|------|
| 主题列表 | `index.html` | 系统入口页，展示所有主题卡片 |
| 主题详情 | `detail.html` | 展示主题下的稿件列表和专题推荐 |
| 稿件填报 | `submit.html` | 填写并提交稿件信息的表单页 |
| 新建主题 | `create-theme.html` | 创建新主题并配置专题推荐 |

### 2.3 页面流转

```
主题列表（入口）
  │
  ├── 点击卡片「详情」按钮 ──→ 主题详情页
  │                               └── 返回按钮 ──→ 主题列表
  │
  ├── 点击卡片「填报」按钮 ──→ 稿件填报页
  │                               └── 返回/提交 ──→ 主题列表
  │
  ├── 点击卡片「编辑」按钮 ──→ 编辑主题页（复用新建主题页，预填已有数据）
  │                               └── 保存 ──→ 主题列表
  │
  └── 点击右上角「+」按钮 ──→ 新建主题页
                                   └── 保存 ──→ 主题列表
```

## 3. 功能需求

### 3.1 主题列表页

**页面路径：** `index.html`（系统入口）

**功能描述：** 展示所有已创建的主题，提供快速导航到详情、填报和新建的入口。

**页面结构：**

| 区域 | 说明 |
|------|------|
| 顶部导航栏 | 固定定位，蓝色渐变背景，左侧显示"主题填报"标题，右侧为金色"+"新建按钮 |
| 内容区域 | 可垂直滚动，展示主题卡片列表 |
| 底部导航栏 | 固定定位，三个 Tab 入口：主题填报（当前高亮）、数据报告（灰色）、异常处置（灰色） |

**主题卡片结构：**

| 元素 | 说明 |
|------|------|
| Banner 区域 | 180px 高，蓝色渐变背景 + 水墨山水 SVG 装饰 |
| 删除按钮 | Banner 右上角，半透明圆形「×」按钮，点击弹出二次确认对话框 |
| 主题标题 | 18px 加粗，最多显示完整标题 |
| 主题描述 | 14px 灰色，最多 2 行，超出省略 |
| 操作按钮 | 三个并排按钮：「详情」（描边样式）、「填报」（填充样式）、「编辑」（灰色幽灵样式，跳转编辑主题页） |

**主题删除交互：**

| 步骤 | 说明 |
|------|------|
| 1. 点击删除按钮 | 阻止事件冒泡，显示确认对话框 |
| 2. 确认对话框 | 居中弹出，标题"确认删除"，提示"删除后不可恢复，确定要删除该主题吗？" |
| 3. 点击取消 | 关闭对话框，无操作 |
| 4. 点击删除 | 卡片淡出 + 缩小动画（0.3s），完成后移除 DOM |
| 5. 点击遮罩 | 关闭对话框 |

**数据字段：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 主题名称 | 文本 | 是 | 显示在卡片标题处 |
| 主题描述 | 文本 | 否 | 显示在卡片描述处，最多 2 行 |
| Banner 图 | 图片 | 是 | 1200×600px，JPG/PNG/WEBP |
| 专题推荐 | 列表 | 否 | 可配置的推荐卡片列表 |

### 3.2 主题详情页

**页面路径：** `detail.html`

**功能描述：** 展示某个主题的完整信息，包括重点稿件列表和专题推荐。

**页面结构：**

| 区域 | 高度/位置 | 说明 |
|------|-----------|------|
| 返回按钮 | 固定悬浮，左上角 | 半透明圆形按钮，返回主题列表 |
| 状态栏 | 44px，顶部固定 | 显示时间和系统图标 |
| Banner 区域 | 280px | 深蓝渐变 + 山水 SVG + 装饰印章元素 |
| Banner 标题 | Banner 内 | 主标题（24px 白色）+ 副标题（32px 金色） |
| 标签栏 | Banner 底部 | 显示警务类型标签，如"共富警务 · 共治警务 · 生态警务" |
| 重点稿件列表 | 可滚动 | 文章卡片，左图右文布局，初始加载 5 条，底部「加载更多」按钮 |
| 稿件删除 | 左滑操作 | 稿件卡片左滑露出红色「删除」区域，点击弹出二次确认 |
| 专题推荐 | 可滚动 | 5 张特色推荐卡片，各有独特配色 |
| 页脚 | 底部 | "浙江省公安厅融媒体中心" |

**重点稿件卡片：**

| 元素 | 说明 |
|------|------|
| 缩略图 | 112×76px，蓝色渐变占位，左上角带序号标签 |
| 稿件标题 | 14px 加粗，最多 2 行，超出省略 |
| 来源 + 时间 | 12px 灰色，格式："来源名 · YYYY/M/D HH:mm" |

**稿件加载更多：**

| 交互 | 说明 |
|------|------|
| 初始加载 | 显示前 5 条稿件 |
| 加载更多按钮 | 稿件列表底部居中，轻量文字链接风格（13px 灰色文字 + 下箭头），点击追加 5 条 |
| 全部加载完 | 按钮自动隐藏 |

**稿件删除交互（左滑删除）：**

| 步骤 | 说明 |
|------|------|
| 1. 左滑卡片 | 稿件卡片向左滑动 80px，露出红色「删除」区域 |
| 2. 点击删除 | 弹出确认对话框，标题"确认删除"，提示"删除后不可恢复，确定要删除该稿件吗？" |
| 3. 确认删除 | 卡片淡出 + 收缩动画（0.3s），完成后移除 DOM |

| 5. 右滑/点击空白 | 收回滑出的卡片 |

| 卡片编号 | 主题 | 配色 |

| 1 | 三大警务体系建设 | 蓝色系 |
| 2 | 三能浙警 | 橙色系 |
| 3 | 百日攻坚专项行动 | 紫色系 |
| 4 | 道路交通秩序综合整治 | 深蓝-青色系 |
| 5 | 藏蓝她声 | 粉色系 |

> **注意：** 当前专题推荐的内容为静态演示数据。实际生产环境中，推荐卡片的内容应从「新建主题」时用户配置的数据中读取。

### 3.3 稿件填报页

**页面路径：** `submit.html`

**功能描述：** 填写并提交稿件信息到指定主题下。

**页面结构：**

| 区域 | 说明 |
|------|------|
| 顶部导航栏 | 白色背景，左侧返回按钮，居中"稿件填报"标题 |
| 基础信息录入卡片 | 白色圆角卡片，包含 4 个表单字段 |
| 封面图上传卡片 | 白色圆角卡片，包含上传区域 |
| 提交按钮 | 蓝色渐变全宽按钮，56px 高 |

**表单字段：**

| 字段 | 类型 | 必填 | 输入方式 | 图标 | 说明 |
|------|------|------|----------|------|------|
| 稿件标题 | 文本 | 是 ★ | 单行输入 | 文档图标 | 占位提示："请输入稿件的完整标题"，标签显示红色「*」 |
| 发布时间 | 日期时间 | 是 ★ | 日期时间选择器 | 日历图标 | 格式：YYYY/MM/DD HH:mm，标签显示红色「*」 |
| 稿件链接 | URL | 是 ★ | 多行文本域 | 链接图标 | 占位提示："https://..."，标签显示红色「*」 |
| 发布媒体 | 文本 | 是 ★ | 单行输入 | 报纸图标 | 占位提示："如：人民网、人民日报、法治日报"，标签显示红色「*」 |

**封面图上传：**

| 属性 | 说明 |
|------|------|
| 必填 | 是 ★，卡片标题显示红色「*」 |
| 上传方式 | 点击上传 |
| 大小限制 | 不大于 2M |
| 支持格式 | jpg、png |
| 上传文案 | 默认显示"点击上传图片"，上传后显示预览 |

**交互行为：**

| 交互 | 说明 |
|------|------|
| 输入聚焦 | 边框变蓝（`#1E4FCC`）+ 外发光效果 |
| 必填校验 | 所有必填字段（稿件标题、发布时间、稿件链接、发布媒体、封面图）为空时显示红色边框 + 错误提示 |
| 提交流程 | 点击提交 → 校验 → 显示"稿件提交成功"Toast → 1.8s 后跳转主题列表 |

### 3.4 新建主题页

**页面路径：** `create-theme.html`

**功能描述：** 创建新的宣传主题，配置主题基本信息和专题推荐内容。

**页面结构：**

| 区域 | 说明 |
|------|------|
| 顶部导航栏 | 白色背景，左侧返回按钮，居中"新建主题"标题 |
| 主题信息卡片 | 白色圆角卡片，包含名称、描述、题图上传 |
| 专题推荐配置卡片 | 白色圆角卡片，可动态添加多个推荐项 |
| 保存按钮 | 蓝色渐变全宽按钮，64px 高 |

**主题信息字段：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 主题名称 | 文本 | 是 ★ | 单行输入，标签显示红色「*」，预填示例："人民公安从这里走来"，最多 20 字 |
| 主题描述 | 多行文本 | 否 | 多行文本域，占位提示："描述该主题的核心精神与报道方向..."，支持自动增高，最多 200 字 |
| 题图上传 | 图片 | 是 ★ | 标签显示红色「*」，支持点击上传，不大于 2M，支持 jpg、png，上传后显示预览覆盖层，可重新上传 |

**专题推荐配置：**

该模块允许用户为当前主题配置专属的专题推荐卡片。每个推荐项对应主题详情页中的一张推荐卡片。

| 操作 | 说明 |
|------|------|
| 添加推荐 | 点击「添加推荐」按钮，动态新增一个推荐项卡片 |
| 删除推荐 | 点击推荐项右上角「×」按钮，带淡出动画移除 |
| 空状态 | 无推荐项时显示引导提示："点击上方按钮添加专题推荐" |

**推荐项字段：**

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 推荐图 | 图片 | 是 ★ | 标签显示红色「*」，不大于 2M，支持 jpg、png |
| 推荐链接 | URL | 是 ★ | 标签显示红色「*」，点击推荐卡片后的跳转地址，占位提示："https://example.com/topic" |

**新增推荐项交互：**

| 交互 | 说明 |
|------|------|
| 添加动画 | 新增项从上方滑入（`slideIn` 动画，0.3s） |
| 自动滚动 | 新增后自动滚动到该项可见位置 |
| 图片上传 | 点击上传区域触发文件选择，选择后实时显示缩略图预览 |
| 删除动画 | 淡出 + 上移（0.2s）后移除 DOM |
| 序号标记 | 自动递增编号："推荐专题 #1"、"推荐专题 #2"... |

**保存交互：**

| 步骤 | 说明 |
|------|------|
| 1. 点击保存 | 按钮显示 loading 旋转动画，文字切换为加载状态 |
| 2. 等待 1s | 模拟保存过程 |
| 3. 显示 Toast | 居中弹出"主题创建成功"提示，带金色对勾动画 |
| 4. 跳转 | 1.6s 后 Toast 消失，自动跳转回主题列表页 |

## 4. 视觉规范

### 4.1 色彩体系

| 角色 | 色值 | 用途 |

## 5. 交互规范

### 5.1 页面过渡

| 交互 | 效果 |
|------|------|
| 卡片点击 | `scale(0.98)`，0.2s |
| 按钮点击 | `opacity: 0.85`，0.2s |
| 返回按钮 | `opacity: 0.6`，0.2s |
| 滚动 | 平滑滚动 `scroll-behavior: smooth` |
| 双击防抖 | iOS 双击缩放已禁用 |

### 5.2 表单交互

| 交互 | 效果 |
|------|------|
| 输入聚焦 | 边框变为 `#1E4FCC` + 蓝色外发光 |
| 标签高亮 | 聚焦时标签字重变粗（400 → 700） |
| 必填校验失败 | 输入框边框变红，下方显示错误提示 |
| 校验恢复 | 输入内容后自动清除错误状态 |
| 文件上传 | 选择文件后显示预览覆盖层 |
| 重新上传 | 点击"重新上传"按钮重新选择文件 |

### 5.3 确认对话框

| 场景 | 标题 | 提示文案 |
|------|------|----------|
| 删除主题 | 确认删除 | 删除后不可恢复，确定要删除该主题吗？ |
| 删除稿件 | 确认删除 | 删除后不可恢复，确定要删除该稿件吗？ |

对话框样式：居中弹出，白色圆角矩形（280px），底部双按钮（取消/删除），点击遮罩关闭。

### 5.4 左滑删除

| 交互 | 效果 |
|------|------|
| 左滑卡片 | 卡片左移 80px，露出红色「删除」区域（0.25s 过渡） |
| 点击删除区域 | 弹出确认对话框 |
| 右滑/点击空白 | 卡片归位 |
| 确认删除 | 卡片淡出 + 收缩动画（0.3s），移除 DOM |

### 5.5 Toast 提示

| 场景 | 文案 | 图标 |

## 7. 数据模型

### 7.1 主题（Theme）

```
Theme {
  id:          string      // 唯一标识
  name:        string      // 主题名称（必填 ★）
  description: string      // 主题描述（非必填）
  bannerImage: file        // 题图（必填 ★，1200×600px）
  createdAt:   datetime    // 创建时间
  recommends:  Recommend[] // 专题推荐列表
}
```

### 7.2 推荐项（Recommend）

```
Recommend {
  id:      string    // 唯一标识
  image:   file      // 推荐图（必填 ★，21:9 比例，≤2MB）
  url:     string    // 推荐链接 URL（必填 ★）
  sortOrder: int     // 排序序号
}
```

### 7.3 稿件（Article）

```
Article {
  id:           string    // 唯一标识
  themeId:      string    // 所属主题 ID
  title:        string    // 稿件标题（必填 ★）
  publishedAt:  datetime  // 发布时间（必填 ★）
  url:          string    // 稿件链接（必填 ★）
  media:        string    // 发布媒体（必填 ★，如：人民网、人民日报）
  coverImage:   file      // 封面图（必填 ★，16:9 比例，≤2MB）
  submittedAt:  datetime  // 提交时间
}
```
