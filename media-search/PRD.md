# 智能媒资首页搜索优化 — PRD

## 产品概述

- **背景**：现有智能媒资系统的搜索能力分散在"精准检索""语义检索""AI 多模态检索"三个独立入口，用户需在不同模式间切换，操作路径长、效率低
- **目标**：将三种搜索能力聚合到首页统一搜索栏，实现"一次搜索、全库呈现"，降低用户使用门槛
- **用户角色**：记者、编辑、素材管理员

---

## 一、首页搜索区域

### 1.1 搜索区域标题

- **展示规则**：
  - 主标题"灵感无界，触手可及"，居中展示，大字号加粗
  - 副标题"海量多模态媒资，AI赋能创意检索"，居中展示，中等字号，浅色文字
  - 主标题和副标题位于搜索栏上方
- **交互规则**：
  - 搜索触发后，标题和副标题始终保持可见，不隐藏
  - 搜索后搜索区域整体缩小上下间距

### 1.2 搜索栏

- **展示规则**：
  - 页面居中展示搜索栏，胶囊形外观，宽度较大营造大气感
  - 默认状态从左到右依次为：搜索图标、文字输入框、图片上传按钮、搜索按钮（文案统一为"AI智搜"）
  - 文字输入框 placeholder 为"输入关键词、自然语言描述搜索全部媒资"
- **交互规则**：
  - 文字输入框获取焦点时，搜索栏阴影增强
  - 按回车键或点击"AI智搜"按钮触发搜索
  - 点击图片上传按钮切换输入模式（见 1.3）
  - 图片上传按钮和搜索按钮始终右对齐
  - 文字输入模式下，搜索框清空后自动返回初始状态（隐藏搜索结果、展开搜索区域、隐藏资源库筛选）

### 1.3 输入模式切换

- **展示规则**：
  - 输入端支持两种互斥模式：文字输入和图片输入
  - 文字模式：展示搜索图标 + 文字输入框 + 图片上传按钮 + "AI智搜"按钮
  - 图片模式：展示图片缩略图预览（右上角带红色 X 删除图标）+ 图片上传按钮 + "AI智搜"按钮，隐藏搜索图标和文字输入框
- **交互规则**：
  - 点击图片上传按钮进入图片模式，上传图片后自动触发搜索，无需手动点击"AI智搜"按钮
  - 点击缩略图右上角 X 图标退出图片模式，恢复文字输入状态，同时清空文字搜索框、清空搜索结果、隐藏资源库筛选，搜索区域恢复为初始展开状态
  - 搜索按钮文案统一为"AI智搜"，不随模式切换变化
  - 在文字搜索结果页中点击图片上传按钮，切换为图片模式并自动搜索
  - 在图片搜索结果页中点击 X 删除图片，退回文字输入模式，清空结果返回初始状态
- **数据规则**：
  - **取数规则**：文字输入模式下读取用户输入的关键词或自然语言描述；图片模式下读取用户上传的图片文件
  - 文字输入支持关键词精确匹配和自然语言语义理解
  - 图片输入基于图片内容进行多模态检索

### 1.4 资源库筛选

- **展示规则**：
  - 搜索前不展示资源库筛选
  - 搜索触发后，搜索栏下方展示资源库筛选按钮，默认选中"全部"
  - 筛选选项：全部、媒资素材库、全省资源库、网络资源库
- **交互规则**：
  - 点击按钮切换资源库筛选，选中按钮高亮（药丸形填充），其余按钮恢复默认样式
  - 资源库筛选对所有类型（图片、视频、稿件）的搜索结果生效
  - 切换资源库后自动刷新搜索结果
  - 返回初始状态时隐藏资源库筛选
- **数据规则**：
  - **数据筛选规则**：根据用户选择的资源库过滤结果，仅返回对应资源库中的资源

## 二、搜索结果页

### 2.1 结果页布局

- **展示规则**：
  - 搜索触发后，搜索区域缩小上下间距，标题和副标题保留
  - 搜索栏下方依次展示资源库筛选、结果列表区域
- **交互规则**：
  - 搜索栏折叠后仍可继续输入或切换模式进行二次搜索
  - 搜索区域缩小过程带有过渡动画

### 2.2 排序规则

- **展示规则**：不展示排序相关的 UI 元素
- **数据规则**：
  - **数据排序规则**：
    - 一级排序：按相似度从高到低排序
    - 二级排序：相似度相同时，按上传时间从近到远排序

### 2.3 结果卡片

- **展示规则**：
  - 搜索结果按图片、视频、稿件（仅文字搜索）分区域展示
  - 图片和视频区域每区域展示一行卡片（5列），最多返回 20 条结果
  - 稿件区域为纯文字列表样式，无缩略图，最多返回 100 条结果
  - 稿件每项展示标题（加粗）、来源和日期（同行，来源左对齐，日期右对齐）
  - 图片卡片：缩略图展示内容预览图
  - 视频卡片：缩略图展示内容预览图，中央叠加半透明圆形播放按钮，右下角展示视频时长标签
- **交互规则**：
  - 卡片 hover 时上浮并显示阴影
  - 稿件列表项 hover 时背景变色
  - 点击卡片或稿件项跳转至资源详情页
  - 图片/视频区域通过左右箭头按钮切换分页，箭头浮于卡片行两侧，垂直居中，第一页隐藏左箭头，最后一页隐藏右箭头
  - 点击"查看更多"携带当前搜索条件和查询模式跳转至搜索中心对应频道：文字搜索模式下，图片区域"查看更多"跳转搜索中心-文搜图，视频区域跳转搜索中心-文搜视频；图片搜索模式下，图片区域跳转搜索中心-图搜图，视频区域跳转搜索中心-图搜视频
- **数据规则**：
  - **取数规则**：根据搜索关键词或图片，从媒资资源库检索匹配的资源；图片/视频返回缩略图、标题、入库时间、来源标签字段，视频额外返回时长字段；稿件返回标题、来源、发布时间字段
  - **数据筛选规则**：根据用户选择的资源库过滤结果
  - **数据排序规则**：各区域内部按相似度从高到低排序，相似度相同时按发布时间从近到远排序
  - 图片和视频结果上限为 20 条，稿件结果上限为 100 条

### 2.4 稿件列表滚动加载

- **展示规则**：
  - 稿件列表为固定高度的可滚动区域，初始展示 5 条
- **交互规则**：
  - 滚动到底部时自动加载更多稿件项（每次加载 5 条）
  - 滚动条样式为细窄型

## 三、顶部导航栏

### 3.1 导航栏结构

- **展示规则**：
  - 导航栏固定在页面顶部，包含：系统 Logo + 名称、主导航菜单、上传按钮、用户头像
  - 系统名称为"智能媒资系统"
  - 导航菜单项：首页（默认选中）、素材中心、资源中心、搜索中心、业务管理
- **交互规则**：
  - 导航菜单项 hover 时文字变色并显示浅蓝背景
  - 点击上传按钮触发上传流程
  - 导航栏不越过 PRD 拖拽分隔线

---

## 四、全局交互规则

- 所有可点击元素具备 hover 状态反馈（颜色变化或位移）
- 搜索区域缩小/展开带有过渡动画
- 卡片类元素 hover 时上浮并增强阴影
- 页脚展示"传播大脑科技（浙江）股份有限公司 · 智能媒资系统"
