# 智能媒资首页搜索优化 — PRD

## 产品概述

- **背景**：现有智能媒资系统的搜索能力分散在"精准检索""语义检索""AI 多模态检索"三个独立入口，用户需在不同模式间切换，操作路径长、效率低
- **目标**：将三种搜索能力聚合到首页统一搜索栏，实现"一次搜索、全库呈现"，降低用户使用门槛
- **用户角色**：记者、编辑、素材管理员

---

## 一、首页搜索区域

### 1.1 搜索区域标题

- **展示规则**：
  - 主标题"灵感无界，触手可及"，居中展示，大字号加粗
  - 副标题"海量多模态媒资，AI赋能创意检索"，居中展示，中等字号，浅色文字
  - 主标题和副标题位于搜索栏上方
- **交互规则**：
  - 搜索触发后，标题和副标题始终保持可见，不隐藏
  - 搜索后搜索区域整体缩小上下间距

### 1.2 搜索栏

- **展示规则**：
  - 页面居中展示搜索栏，胶囊形外观，宽度较大营造大气感
  - 默认状态从左到右依次为：搜索图标、文字输入框、图片上传按钮、搜索按钮（文案为"语义检索"）
  - 文字输入框 placeholder 为"输入关键词、自然语言描述搜索全部媒资"
- **交互规则**：
  - 文字输入框获取焦点时，搜索栏阴影增强
  - 按回车键或点击"语义检索"按钮触发搜索
  - 点击图片上传按钮切换输入模式（见 1.3）
  - 图片上传按钮和搜索按钮始终右对齐

### 1.3 输入模式切换

- **展示规则**：
  - 输入端支持两种互斥模式：文字输入和图片输入
  - 文字模式：展示搜索图标 + 文字输入框 + 图片上传按钮 + 搜索按钮
  - 图片模式：展示图片缩略图预览（右上角带红色 X 删除图标）+ 图片上传按钮 + "多模态检索"按钮，隐藏搜索图标和文字输入框
- **交互规则**：
  - 点击图片上传按钮进入图片模式
  - 点击缩略图右上角 X 图标退出图片模式，恢复文字输入状态
  - 图片模式下搜索按钮文案变为"多模态检索"
- **数据规则**：
  - **取数规则**：文字输入模式下读取用户输入的关键词或自然语言描述；图片模式下读取用户上传的图片文件
  - 文字输入支持关键词精确匹配和自然语言语义理解
  - 图片输入基于图片内容进行多模态检索

### 1.4 搜索类型筛选

- **展示规则**：搜索栏下方展示三个药丸形按钮：全部（默认选中）、图片、视频
- **交互规则**：
  - 点击按钮切换筛选类型，选中按钮高亮，其余按钮恢复默认样式
  - 类型筛选仅影响搜索结果的内容类型过滤，不改变输入模式（文字/图片）
  - 若已有搜索结果展示中，切换类型后自动刷新结果
- **数据规则**：
  - **数据筛选规则**：
    - 全部：返回图片和视频两种类型的结果
    - 图片：仅返回图片类型结果
    - 视频：仅返回视频类型结果
  - 多模态检索当前仅支持图片和视频，不支持音频和文档

## 二、搜索结果页

### 2.1 结果页布局

- **展示规则**：
  - 搜索触发后，搜索区域缩小上下间距，标题和副标题保留
  - 搜索栏下方展示结果列表区域
  - 结果列表顶部左侧展示"共找到 X 条相关结果"
- **交互规则**：
  - 搜索栏折叠后仍可继续输入或切换模式进行二次搜索
  - 搜索区域缩小过程带有过渡动画

### 2.2 排序规则

- **展示规则**：不展示排序相关的 UI 元素
- **数据规则**：
  - **数据排序规则**：
    - 一级排序：按相似度从高到低排序
    - 二级排序：相似度相同时，按上传时间从近到远排序

### 2.3 结果卡片

- **展示规则**：
  - 以宫格视图展示搜索结果，每行 5 列（大屏）
  - 每个卡片包含：缩略图区域、类型标签（图片/视频）、标题、作者、日期、标签
  - 缩略图区域用颜色区分内容类型：图片蓝色系、视频橙色系
- **交互规则**：
  - 卡片 hover 时上浮并显示阴影
  - 点击卡片跳转至资源详情页
- **数据规则**：
  - **取数规则**：根据搜索关键词或图片，从媒资资源库检索匹配的资源，返回缩略图、标题、作者、入库日期、标签字段
  - **数据筛选规则**：根据用户选择的类型（全部/图片/视频）过滤结果
  - **数据排序规则**：按 2.2 排序规则执行

### 2.4 分页

- **展示规则**：结果列表下方展示分页组件，包含页码按钮和下一页按钮，当前页高亮
- **交互规则**：
  - 点击页码切换页面
  - 页码切换后结果列表滚动至顶部

## 三、顶部导航栏

### 3.1 导航栏结构

- **展示规则**：
  - 导航栏固定在页面顶部，包含：系统 Logo + 名称、主导航菜单、上传按钮、用户头像
  - 导航菜单项：首页（默认选中）、素材中心、资源中心、搜索中心、业务管理
- **交互规则**：
  - 导航菜单项 hover 时文字变色并显示浅蓝背景
  - 点击上传按钮触发上传流程
  - 导航栏不越过 PRD 拖拽分隔线

---

## 四、全局交互规则

- 所有可点击元素具备 hover 状态反馈（颜色变化或位移）
- 搜索区域缩小/展开带有过渡动画
- 卡片类元素 hover 时上浮并增强阴影
- 页脚展示"传播大脑科技（浙江）股份有限公司 · 宁报智能媒资系统"
