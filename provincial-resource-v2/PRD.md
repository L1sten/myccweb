# 全省资源库 v2 PRD

## 产品概述

- **背景**：现有首页搜索、资源统计和全省资源库分布在不同页面和模块中。资源库入口依赖资源中心下拉菜单，用户需要先识别层级关系再进入稿件、图片、视频、音频等子模块，路径偏长。
- **目标**：将首页搜索、最新素材、资源入库统计与全省资源库主流程合并为统一入口和工作流，突出全省优质成品内容的发现、检索、筛选和取用效率。
- **用户角色**：
  - 省级媒体编辑：快速发现全省优质素材和成品稿件，完成选题、复用和取稿。
  - 市县融媒体编辑：查看全省内容动态，了解同题报道和优质素材。
  - 资源运营人员：观察入库规模、内容结构和趋势，辅助运营调度。

## 一、顶部导航与页面入口

### 1.1 顶部主导航

- **展示规则**：
  - 顶部主导航沿用线上版本已存在的导航布局，包含：首页、搜索中心、资源中心、素材中心、审批中心、同步中心、标签中心。
  - “全省资源库”为一级独立入口，不再放入“资源中心”下拉菜单，并追加在现有线上导航之后。
  - 当前页面对应主导航项展示选中状态。
- **交互规则**：
  - 点击“首页”展示首页工作台。
  - 点击“全省资源库”进入全省资源库聚合页。
  - 其他主导航项保留可点击状态，用于表达系统完整导航，但不触发复杂跳转。
- **数据规则**：
  - **取数规则**：主导航从系统模块配置读取可见模块。
  - **数据筛选规则**：按当前用户权限展示可访问模块；无权限模块不展示。
  - **数据排序规则**：按线上现有模块顺序展示：首页、搜索中心、资源中心、素材中心、审批中心、同步中心、标签中心、全省资源库。

### 1.2 全省资源库资源类型 Tab

- **展示规则**：
  - 全省资源库页顶部首层 Tab 展示成品稿件、成品图片、成品视频、成品音频四类资源。
  - 该资源类型 Tab 占用原“客户端”渠道 Tab 的位置。
  - 默认选中“成品稿件”。
- **交互规则**：
  - 点击不同资源类型后，搜索条件、筛选栏和结果区按资源类型刷新。
  - 成品稿件使用列表顶部紧凑筛选与 AI智搜工具栏；成品图片、成品视频、成品音频使用日期范围筛选、多模态 AI智搜工具栏和结果操作条。
- **数据规则**：
  - **取数规则**：从全省资源库读取对应资源类型的成品内容。
  - **数据筛选规则**：资源类型为单选条件，切换后仅展示当前类型资源。
  - **数据排序规则**：默认按发布时间倒序。

### 1.3 客户端固定筛选条件

- **展示规则**：
  - 页面不再展示“客户端”渠道 Tab 行。
  - 客户端作为全省资源库的固定筛选条件，不占用顶部可视导航层级。
  - 本期不展示网站、个人创作中心、全部渠道等其他渠道入口。
- **交互规则**：
  - 用户无需切换渠道，资源类型切换、搜索和地域筛选均默认在客户端渠道范围内生效。
  - 页面顶部仅保留成品稿件、成品图片、成品视频、成品音频资源类型 Tab。
- **数据规则**：
  - **取数规则**：读取发布渠道为客户端的全省资源库内容。
  - **数据筛选规则**：客户端渠道作为固定筛选条件，与资源类型、关键词、发布时间、地域等条件取交集。
  - **数据排序规则**：渠道固定后不改变当前排序方式。

## 二、首页优化

### 2.1 线上已有首页模块

- **展示规则**：
  - 首页已上线模块保持现有展示，不在本 PRD 中重新定义字段、样式和交互细节。
  - 原有模块包括首页搜索区、最新图片资源、资源聚合、资源入库趋势等。
  - Web 首页搜索框下不展示资源库范围或资源类型 tab；资源库范围 tab 仅在搜索结果页展示，沿用线上已有逻辑。
  - 线上搜索结果页资源范围 tab 中，原“媒资素材库”命名调整为“素材中心”。
  - 首页在“最新图片资源”上方展示资源范围 tab，包含“本地资源库”和“全省资源库”。
  - “本地资源库”tab 下展示最新图片资源、资源聚合、资源入库趋势。
  - “全省资源库”tab 下展示最新素材、全省资源库数据统计、入库统计趋势。
  - 本期优化“资源聚合”的数据统计展示，图片、视频、音频数据卡片展示资源总量和当前统计周期增量。
  - 无全省资源库权限时，首页仅展示线上已有模块，整体顺序和默认状态保持不变。
- **交互规则**：
  - 原有模块沿用线上交互规则。
  - 用户同时具备本地资源库和全省资源库权限时，首页默认选中“本地资源库”tab；点击 tab 后切换对应模块组。
  - 用户切换“资源入库趋势”统计周期时，同步刷新“资源聚合”数据卡片的周期增量和趋势图；统计周期包括近两周、近一个月、近三个月。
- **数据规则**：
  - **取数规则**：资源聚合总量沿用线上当前素材中心累计统计口径；周期增量读取当前素材中心中按入库时间落在所选统计周期内的新增图片、视频、音频素材。
  - **数据筛选规则**：仅统计当前用户可见、已入库且状态正常的公共媒资素材；下架、删除、权限不可见资源不纳入统计。
  - **数据排序规则**：趋势图按统计周期时间正序展示；数据卡片按图片、视频、音频固定顺序展示。
  - **数据聚合规则**：近两周增量为“资源入库趋势”近两周视图中最近 14 个统计日的新增入库量求和；近一个月增量为近一个月视图中各周新增入库量求和；近三个月增量为近三个月视图中各月新增入库量求和。

### 2.2 最新素材

- **展示规则**：
  - “最新素材”为首页新增模块，展示在“全省资源库”tab 内。
  - 模块展示全省资源库最新入库的视频或图片卡片，卡片包含封面、资源类型、时长或图片数量、标题、来源和发布时间。
  - 卡片底部不展示“专题热度高”“适合图集复用”等额外理由文案，避免首页信息噪声过重。
  - 无全省资源库权限时不展示该模块，也不展示“全省资源库”tab。
- **交互规则**：
  - 点击素材卡片进入对应资源详情；资源详情暂未接入时跳转至全省资源库对应列表，并保留当前资源类型。
  - 点击“进入全省资源库”进入全省资源库聚合页，并保留当前默认资源类型。
  - 模块不影响首页线上已有模块的点击、搜索和统计交互。
- **数据规则**：
  - **取数规则**：从全省资源库中读取当前用户有权限查看、发布渠道为客户端、状态为已入库且可复用的图片和视频资源；默认取近 30 天入库资源作为候选池。
  - **字段排除规则**：自动排除标题为空、正文/说明为空、封面为空、来源为空、发布时间为空、文件大小为 0、宽高缺失、图片宽或高低于展示阈值、视频时长缺失或为 0 的素材。文件格式为 gif、animated webp、apng 的资源不进入最新素材展示。
  - **数据排序规则**：候选素材按字段完整度分层展示：优先展示标题、正文/说明、来源、发布时间、封面、格式、大小、宽高或时长均完整的素材；同层内按发布时间倒序。排序后按资源类型、来源和地域做轮询重排，避免 4 张卡片全部来自同一类型、同一来源或同一地域。
  - **数据去重规则**：同一稿件ID下的多条图片或视频资源只保留发布时间最新的一条；无稿件ID的资源不参与该去重规则。
  - **数据兜底规则**：若筛选后不足 4 条，放宽正文/说明非空要求；不得放宽权限、封面、格式、大小、宽高、时长等硬条件。仍不足时按实际条数展示，不展示空卡片；无可展示素材时隐藏该模块。

### 2.3 全省资源库数据统计

- **展示规则**：
  - “全省资源库数据统计”为首页新增模块，展示在“全省资源库”tab 内，位于最新素材之后。
  - 模块标题下不展示副标题说明。
  - 模块展示稿件、图片、视频、音频的入库总量、周期增量和入库统计趋势图。
  - 模块不替换线上已有“资源入库趋势”，两者并列表达不同数据口径。
  - 无全省资源库权限时隐藏该模块，首页底部保持线上已有模块收尾。
- **交互规则**：
  - 用户切换统计周期时，仅刷新“全省资源库数据统计”模块内的指标卡周期增量和入库统计趋势图。
  - 周期切换不影响线上已有“资源入库趋势”的周期选择和展示状态。
  - 鼠标悬停趋势图节点时展示对应日期或周期的稿件、图片、视频、音频入库数量。
- **数据规则**：
  - **取数规则**：按入库时间读取当前用户权限范围内全省资源库稿件、图片、视频、音频四类资源的新增入库数据。
  - **数据筛选规则**：仅统计已入库、状态正常且用户有权限查看的资源；下架、删除、审核未通过和权限不可见资源不纳入统计。
  - **数据排序规则**：趋势图按统计周期的时间正序展示；指标卡按稿件、图片、视频、音频固定顺序展示。
  - **数据聚合规则**：按日、周或月聚合不同周期内的入库量，同一资源多次更新不重复计入新增入库量。

### 2.4 权限展示规则

- **展示规则**：
  - 首页资源范围由素材中心权限和全省资源库权限共同决定。
  - 用户同时具备素材中心权限和全省资源库权限时，展示“本地资源库”“全省资源库”两个 tab，默认选中“本地资源库”。
  - 用户具备全省资源库权限但不具备素材中心权限时，仅展示全省资源库内容，资源范围 tab 隐藏。
  - 用户具备素材中心权限但不具备全省资源库权限时，仅展示本地资源库内容，资源范围 tab 隐藏。
  - 权限差异不影响当前可见模块内部的展示、顺序和交互。
- **交互规则**：
  - 账号权限由系统权限配置决定，权限状态变化后按最新权限刷新首页可见内容。
  - 权限变更后，下次进入首页按最新权限重新判断是否展示新增模块。
  - 无权限用户通过首页搜索命中全省资源库内容时，仍按既有权限规则过滤不可见资源。
- **数据规则**：
  - **取数规则**：权限状态从用户角色、素材中心权限和全省资源库权限配置读取。
  - **数据筛选规则**：首页仅返回当前租户具备权限的资源域数据；不具备权限的资源域不展示 tab、不展示模块、不返回推荐和统计数据。
  - **数据排序规则**：权限判断不改变已有模块的数据排序。

### 2.5 移动端首页优化

- **展示规则**：
  - 移动端首页沿用“灵感无界，触手可及”和“海量多模态媒资，AI赋能创意检索”的首页表达，展示 AI智搜、图片/视频类型切换、搜索框和最新素材。
  - 移动端首页仅展示一个最新素材模块，不展示素材中心/全省资源库切换。
  - 搜索列表页展示全省资源库/素材中心切换，默认选中全省资源库，素材中心展示在全省资源库之后；移动端首页态不展示该切换。

- **交互规则**：
  - AI智搜支持自然语言描述和图片理解入口，不提供精准检索模式切换。
  - 搜索结果页点击左上角返回按钮后返回移动端首页态。
  - 在搜索列表页切换全省资源库/素材中心时刷新搜索结果列表，并保留当前图片/视频类型和输入内容。
  - 切换图片/视频类型时刷新最新素材和搜索结果列表。
- **数据规则**：
  - **取数规则**：复用 Web 首页多模态检索口径和用户权限口径；移动端首页最新素材由系统按权限判定后切换数据来源：有全省资源库权限时读取全省资源库最新素材，没有全省资源库权限时读取素材中心最新素材；搜索列表页选择素材中心时读取线上首页原有可检索媒资，选择全省资源库时读取当前用户可见的全省资源库成品图片和成品视频。
  - **数据筛选规则**：按最新素材区或搜索列表页所在场景、选中的资源类型、关键词或自然语言意图、用户权限取交集；移动端首页最新素材区不做资源库切换展示，只返回权限判定后的单一资源域素材。
  - **权限过滤规则**：移动端搜索列表页仅展示当前租户有权限的资源库范围；无权限的资源库范围不展示切换项、不返回搜索结果。
  - **数据排序规则**：移动端首页最新素材区有全省资源库权限时按全省资源库首页最新素材排序口径展示，无全省资源库权限时按素材中心发布时间倒序展示最新素材；搜索结果按检索相关性优先，同等相关性下按发布时间倒序展示。

## 三、全省资源库搜索升级

### 3.1 全省资源库列表页检索布局

- **展示规则**：
  - 全省资源库列表页不使用首页大搜索条，统一使用列表页紧凑检索布局。
  - 检索区域采用列表顶部筛选区在前、AI智搜搜索框在后的横向布局。
  - 右侧搜索框展示搜索图标、关键词/自然语言输入框和“AI智搜”按钮。
  - 普通进入全省资源库列表页时，不展示“全部/素材中心/全省资源库/网络资源库”范围 Tab；该范围 Tab 仅在首页搜索进入的搜索结果页展示。
- **交互规则**：
  - 点击“AI智搜”或按回车后，按输入内容刷新当前资源类型结果。
  - 切换成品稿件、成品图片、成品视频、成品音频时，筛选模块和搜索框按资源类型刷新。
  - 首页搜索进入结果页时，可通过范围 Tab 切换全部、素材中心、全省资源库、网络资源库；直接进入全省资源库列表页时不出现该范围 Tab。
- **数据规则**：
  - **取数规则**：读取当前资源类型、关键词/自然语言描述、发布时间、地域和资源类型对应的筛选条件。
  - **数据筛选规则**：输入内容与资源类型、客户端渠道、地域、发布时间和当前资源类型筛选条件取交集。
  - **数据排序规则**：默认按发布时间倒序。

### 3.2 成品稿件筛选区域

- **展示规则**：
  - 成品稿件列表顶部筛选区默认展示，不再放入搜索框内折叠。
  - 筛选模块展示为单个日期范围控件和稿件类型下拉控件。
  - 日期范围控件默认展示“创建开始日期 至 创建结束日期”。
  - 稿件类型下拉默认展示“全部稿件”。
  - 搜索框内不展示图搜入口、筛选按钮、精准检索/语义检索模式 Tab、字段下拉、匹配方式或追加条件入口。
- **交互规则**：
  - 用户可点击日期范围控件展开日期面板，并调整开始日期和结束日期。
  - 用户可通过稿件类型下拉切换全部稿件、新闻、图集、视频、音频、链接、视频专辑、音频专辑、专题、活动、直播。
  - 切换日期范围或稿件类型后刷新成品稿件结果，并清空当前勾选状态。
  - 右侧搜索框只承担关键词或自然语言检索动作。
- **数据规则**：
  - **取数规则**：读取当前发布时间开始日期、发布时间结束日期和稿件类型条件。
  - **数据筛选规则**：发布时间闭区间、稿件类型、搜索输入、资源类型、客户端渠道和地域取交集；选择全部稿件时不限制稿件类型。
  - **数据排序规则**：不改变当前排序方式。

### 3.3 发布时间筛选

- **展示规则**：
  - 成品稿件、成品图片、成品视频、成品音频列表页上方均展示发布时间筛选。
  - 成品稿件、成品图片、成品视频、成品音频均使用单个日期范围控件。
  - 日期范围控件默认展示“创建开始日期 至 创建结束日期”。
  - 日期范围选择器下拉面板包含今天、昨天、最近三天、最近一周、最近一个月快捷项，以及月份日历、年份选择和月份选择入口。
- **交互规则**：
  - 点击日期范围控件时展开日期范围面板。
  - 点击快捷项后自动在同一个日期范围控件内回填开始日期和结束日期。
  - 点击日历日期后按所选日期回填日期范围；再次选择可更新日期条件。
- **数据规则**：
  - **取数规则**：读取资源发布时间。
  - **数据筛选规则**：成品稿件、成品图片、成品视频、成品音频均按开始日期和结束日期闭区间筛选资源。
  - **数据排序规则**：不改变当前排序方式。

### 3.4 稿件类型

- **展示规则**：
  - 稿件类型包含新闻、图集、视频、音频、链接、视频专辑、音频专辑、专题、活动、直播。
  - 稿件类型以下拉控件形式仅在成品稿件筛选区域中展示，默认项为全部稿件。
- **交互规则**：
  - 稿件类型单选，切换后立即参与筛选。
- **数据规则**：
  - **取数规则**：读取资源内容类型。
  - **数据筛选规则**：按所选稿件类型过滤；选择全部稿件时不限制稿件类型。
  - **数据排序规则**：不改变当前排序方式。

### 3.5 图片、视频、音频多模态搜索

- **展示规则**：
  - 成品图片、成品视频、成品音频不展示稿件类型筛选。
  - 列表顶部筛选区展示日期范围选择器，右侧展示紧凑多模态 AI智搜搜索框。
  - 搜索框展示搜索图标、关键词/自然语言输入框和“AI智搜”按钮。
  - 成品图片、成品视频、成品音频均不展示图搜入口或上传图片入口。
  - 输入框统一使用“输入关键词、自然语言描述搜索全部媒资”占位文案，不按成品图片、成品视频、成品音频额外区分资源库。
  - 该模块不展示首页下方的素材中心、全省资源库、网络资源库筛选 chip；仅首页搜索进入的结果页展示范围 Tab。
  - 批量选择栏移动到结果区内部、卡片网格或音频列表上方，包含全选、已选数量和清除入口。
- **交互规则**：
  - 点击“AI智搜”或按回车后，按输入内容刷新当前资源类型结果。
  - 选择日期范围或快捷日期范围后刷新当前资源类型结果。
  - 勾选全选时选中当前页全部资源；取消全选时取消当前页全部选择。
  - 点击清除时取消当前页已选资源，并将已选数量恢复为 0。
- **数据规则**：
  - **取数规则**：读取当前资源类型、关键词/自然语言描述、发布时间开始日期、发布时间结束日期和当前页选择状态。
  - **数据筛选规则**：文本语义、发布时间闭区间、地域和客户端渠道取交集；当前资源类型仅取成品图片、成品视频或成品音频。
  - **数据排序规则**：默认按多模态相关度倒序；相关度相同时按发布时间倒序。

## 四、全省资源库结果区

### 4.1 左侧地域筛选

- **展示规则**：
  - 左侧地域筛选按省本级和地市分组展示。
  - 地市分组可展开区县和市本级。
  - 地域节点不展示数量。
  - 已选地域展示重置入口。
- **交互规则**：
  - 勾选地市时，默认选中该地市下全部子项。
  - 勾选区县时，仅筛选该区县内容。
  - 重置入口可一次性恢复地域筛选默认状态。
- **数据规则**：
  - **取数规则**：根据资源发布站点对应的地域属性读取省、市、区县层级。
  - **数据筛选规则**：多地域取并集，与资源类型、渠道和搜索条件取交集。
  - **数据排序规则**：不影响结果排序。

### 4.2 成品稿件结果

- **展示规则**：
  - 成品稿件以表格展示，包含选择框、序号、稿件标题、来源、稿件类型、字数、发布站点、发布时间、操作。
  - 标题可点击进入详情。
  - 列表上方展示批量取稿按钮和当前结果数量。
- **交互规则**：
  - 支持单选、全选和批量取稿。
  - 点击单行“取稿”弹出取稿渠道选择。
  - 切换筛选条件后清空当前勾选状态。
- **数据规则**：
  - **取数规则**：读取全省资源库稿件标题、来源、类型、字数、发布站点、发布时间和可取稿状态。
  - **数据筛选规则**：按资源类型、渠道、地域、发布时间、稿件类型和关键词取交集。
  - **数据排序规则**：默认按发布时间倒序。

### 4.3 成品图片、成品视频、成品音频结果

- **展示规则**：
  - 成品图片和成品视频以卡片网格展示；成品音频以列表展示。
  - 成品图片卡片展示选择框、封面、标题、来源和发布时间。
  - 成品视频卡片展示选择框、封面、播放标识、时长、标题、来源和发布时间。
  - 成品音频列表展示选择框、标题、来源、时长、格式、发布时间和操作。
  - 结果区上方展示批量选择栏；日期范围筛选和紧凑多模态 AI智搜搜索框位于结果区上方的检索工具栏。
- **交互规则**：
  - 点击卡片或标题进入资源详情。
  - 图片、视频支持添加到个人素材库。
  - 音频支持播放预览和下载。
- **数据规则**：
  - **取数规则**：读取对应资源类型的标题、封面或预览、来源、发布时间、时长、格式等业务字段。
  - **数据筛选规则**：按资源类型、渠道、地域、发布时间和关键词取交集。
  - **数据排序规则**：默认按发布时间倒序。

### 4.4 成品图片、成品视频、成品音频详情页

- **展示规则**：
  - 点击成品图片、成品视频、成品音频结果后进入详情页，详情页保持线上智能媒资系统的左右分栏结构。
  - 详情页顶部展示操作条，提供“添加到个人素材库”和“下载”两个主操作。
  - 全省资源库成品图片、成品视频、成品音频详情页不展示“上一个”“下一个”跳转按钮。
  - “上一个”“下一个”仅在个人素材库、部门素材库、公共素材库的素材详情页中展示。
  - 成品图片详情左侧展示大图预览，下方展示大小、格式、尺寸；右侧展示“基本信息 / 下载记录”页签，默认展示基本信息。
  - 成品视频详情左侧展示视频预览区、播放按钮、封面图区域和视频数据；右侧展示“基本信息 / 下载记录”页签，默认展示基本信息。
  - 成品音频详情左侧展示音频播放器、基本信息和音频数据；右侧默认展示下载记录表格，暂无下载记录时展示空状态。
- **交互规则**：
  - 点击“添加到个人素材库”后反馈添加结果。
  - 点击“下载”后触发下载动作。
  - 个人素材库、部门素材库、公共素材库详情页中，点击“上一个”切换到当前素材上下文中的上一条素材，点击“下一个”切换到下一条素材；首条素材的“上一个”和末条素材的“下一个”置灰不可点。
  - 若当前素材属于某个文件夹，“上一个”“下一个”仅在该文件夹内切换，不跨文件夹、不跳转到素材库根目录或其他文件夹。
  - 若当前素材不属于任何文件夹，“上一个”“下一个”按进入详情页时的当前素材列表上下文切换。
  - 详情页与列表页保持同一资源类型上下文，返回全省资源库时仍停留在进入详情前的成品类型。
- **数据规则**：
  - **取数规则**：详情页读取素材标题、发布时间、关联稿件标题、渠道、发布站点、库名称、大小、格式、尺寸、时长、信息 ID 等已有字段。
  - **数据筛选规则**：详情页只展示用户在列表中可见且有权限访问的素材；“上一个”“下一个”仅取个人素材库、部门素材库、公共素材库中的可见素材，且与当前素材所属素材库范围一致。
  - **数据排序规则**：个人素材库、部门素材库、公共素材库详情页的“上一个”“下一个”按进入详情页时当前素材列表的排序顺序跳转；素材属于文件夹时，排序范围限定为该文件夹内素材；详情页不改变列表排序；下载记录按下载时间倒序展示。

### 4.5 分页

- **展示规则**：
  - 结果区底部展示总数、每页条数、页码、上一页、下一页和跳转页码。
- **交互规则**：
  - 点击页码切换页面。
  - 每页条数切换后回到第一页。
  - 页码越界时提示并保持当前页。
- **数据规则**：
  - **取数规则**：按当前筛选条件和页码截取结果。
  - **数据筛选规则**：分页不改变筛选条件。
  - **数据排序规则**：分页保持当前排序方式。

## 五、全局交互规则

- 所有可点击控件具备 hover、focus 和 active 状态。
- 搜索、筛选、资源类型切换、渠道切换均需提供即时视觉反馈。
- 无结果时展示空状态，提示用户调整关键词、时间或地域筛选。
- 统计图、最新素材、资源列表中的数据仅展示当前用户权限范围内可见内容。
- 全省资源库页面视觉按政务后台风格收敛，使用白底浅边框、紧凑表单、低阴影和明确功能色，避免装饰性渐变和过重视觉元素。
