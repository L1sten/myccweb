# 全省资源库 v2 PRD

## 产品概述

- **背景**：现有首页搜索、资源统计和全省资源库分布在不同原型中。资源库入口依赖资源中心下拉菜单，用户需要先识别层级关系再进入稿件、图片、视频、音频等子模块，路径偏长。
- **目标**：将首页搜索、推荐素材、资源入库统计与全省资源库主流程合并为一个可评审原型，突出全省优质成品内容的发现、检索、筛选和取用效率。
- **用户角色**：
  - 省级媒体编辑：快速发现全省优质素材和成品稿件，完成选题、复用和取稿。
  - 市县融媒体编辑：查看全省内容动态，了解同题报道和优质素材。
  - 资源运营人员：观察入库规模、内容结构和趋势，辅助运营调度。

## 一、顶部导航与页面入口

### 1.1 顶部主导航

- **展示规则**：
  - 顶部主导航包含：首页、全省资源库、素材中心、资源中心、搜索中心、业务管理。
  - “全省资源库”为一级独立入口，不再放入“资源中心”下拉菜单。
  - 当前页面对应主导航项展示选中状态。
- **交互规则**：
  - 点击“首页”展示首页工作台。
  - 点击“全省资源库”进入全省资源库聚合页。
  - 其他主导航项保留可点击状态，用于表达系统完整导航，但不触发复杂跳转。
- **数据规则**：
  - **取数规则**：主导航从系统模块配置读取可见模块。
  - **数据筛选规则**：按当前用户权限展示可访问模块；无权限模块不展示。
  - **数据排序规则**：按后台配置的模块顺序展示，首页固定在首位，全省资源库固定在首页之后。

### 1.2 全省资源库资源类型 Tab

- **展示规则**：
  - 全省资源库页顶部首层 Tab 展示成品稿件、成品图片、成品视频、成品音频四类资源。
  - 该资源类型 Tab 占用原“客户端”渠道 Tab 的位置。
  - 默认选中“成品稿件”。
- **交互规则**：
  - 点击不同资源类型后，搜索条件、筛选栏和结果区按资源类型刷新。
  - 成品稿件使用新版精准检索和语义检索模块；成品图片、成品视频、成品音频使用旧版日期范围、关键词和批量选择筛选模块。
- **数据规则**：
  - **取数规则**：从全省资源库读取对应资源类型的成品内容。
  - **数据筛选规则**：资源类型为单选条件，切换后仅展示当前类型资源。
  - **数据排序规则**：默认按发布时间倒序；语义检索模式下优先按相关度倒序，再按发布时间倒序。

### 1.3 客户端固定筛选条件

- **展示规则**：
  - 页面不再展示“客户端”渠道 Tab 行。
  - 客户端作为全省资源库的固定筛选条件，不占用顶部可视导航层级。
  - 本期不展示网站、个人创作中心、全部渠道等其他渠道入口。
- **交互规则**：
  - 用户无需切换渠道，资源类型切换、搜索和地域筛选均默认在客户端渠道范围内生效。
  - 页面顶部仅保留成品稿件、成品图片、成品视频、成品音频资源类型 Tab。
- **数据规则**：
  - **取数规则**：读取发布渠道为客户端的全省资源库内容。
  - **数据筛选规则**：客户端渠道作为固定筛选条件，与资源类型、关键词、发布时间、地域等条件取交集。
  - **数据排序规则**：渠道固定后不改变当前排序方式。

## 二、首页优化

### 2.1 线上已有首页模块

- **展示规则**：
  - 首页已上线模块保持现有展示，不在本 PRD 中重新定义字段、样式和交互细节。
  - 原有模块包括首页搜索区、最新图片资源、资源聚合、资源入库趋势等。
  - 无全省资源库权限时，首页仅展示线上已有模块，整体顺序和默认状态保持不变。
- **交互规则**：
  - 原有模块沿用线上交互规则。
  - 本期不改变原有首页搜索、最新图片资源、资源聚合和资源入库趋势的使用方式。
- **数据规则**：
  - **取数规则**：原有模块沿用线上取数口径。
  - **数据筛选规则**：原有模块沿用线上筛选口径。
  - **数据排序规则**：原有模块沿用线上排序口径。

### 2.2 全省资源库推荐素材

- **展示规则**：
  - “全省资源库推荐素材”为首页新增模块，展示在“最新图片资源”之后、“资源聚合”之前。
  - 模块展示优质视频或图片卡片，卡片包含封面、资源类型、时长或图片数量、标题、来源和发布时间。
  - 卡片底部不展示“专题热度高”“适合图集复用”等推荐理由文案，避免首页信息噪声过重。
  - 无全省资源库权限时隐藏该模块，线上已有“资源聚合”自然上移。
- **交互规则**：
  - 点击素材卡片进入对应资源详情；原型阶段可先跳转至全省资源库页。
  - 点击“进入全省资源库”进入全省资源库聚合页，并保留当前默认资源类型。
  - 模块不影响首页线上已有模块的点击、搜索和统计交互。
- **数据规则**：
  - **取数规则**：从全省资源库中读取当前用户有权限查看、状态为已入库且可复用的图片和视频资源；优先读取运营推荐池中的资源，推荐池不足时从近 30 天高质量资源候选池补足。
  - **数据筛选规则**：仅纳入发布渠道为客户端、资源状态正常、封面可用、标题非空、未下架且未被权限过滤的素材；图片需具备可展示缩略图，视频需具备封面和时长。
  - **数据排序规则**：先按人工置顶/运营推荐权重倒序；权重相同时按资源质量分倒序；仍相同时按近 7 天复用量、收藏量或点击量综合热度倒序；最后按发布时间倒序兜底。
  - **数据去重规则**：同一专题、同一稿件拆出的多张图片或同一视频多版本只保留排序最高的一条，避免首页重复展示相近素材。
  - **数据兜底规则**：推荐池不足 4 条时，按上述筛选和排序规则从全省资源库候选池补齐；仍不足时按实际条数展示，不展示空卡片。

### 2.3 全省资源库数据统计

- **展示规则**：
  - “全省资源库数据统计”为首页新增模块，展示在已有“资源入库趋势”之后，作为首页底部补充统计区域。
  - 模块展示稿件、图片、视频、音频的入库总量、周期增量和入库统计趋势图。
  - 模块不替换线上已有“资源入库趋势”，两者并列表达不同数据口径。
  - 无全省资源库权限时隐藏该模块，首页底部保持线上已有模块收尾。
- **交互规则**：
  - 用户切换统计周期时，仅刷新“全省资源库数据统计”模块。
  - 周期切换不影响线上已有“资源入库趋势”的周期选择和展示状态。
  - 鼠标悬停趋势图节点时展示对应日期或周期的稿件、图片、视频、音频入库数量。
- **数据规则**：
  - **取数规则**：按入库时间读取当前用户权限范围内全省资源库稿件、图片、视频、音频四类资源的新增入库数据。
  - **数据筛选规则**：仅统计已入库、状态正常且用户有权限查看的资源；下架、删除、审核未通过和权限不可见资源不纳入统计。
  - **数据排序规则**：趋势图按统计周期的时间正序展示；指标卡按稿件、图片、视频、音频固定顺序展示。
  - **数据聚合规则**：按日、周或月聚合不同周期内的入库量，同一资源多次更新不重复计入新增入库量。

### 2.4 权限展示规则

- **展示规则**：
  - 用户具备全省资源库权限时，首页展示“全省资源库推荐素材”和“全省资源库数据统计”两个新增模块。
  - 用户不具备全省资源库权限时，首页不展示上述两个新增模块，默认展示线上已有首页内容。
  - 权限差异不影响线上已有首页模块的展示、顺序和交互。
- **交互规则**：
  - 原型中的“账号权限”切换仅用于评审展示有权限和无权限两种首页状态，不作为开发需求。
  - 权限变更后，下次进入首页按最新权限重新判断是否展示新增模块。
  - 无权限用户通过首页搜索命中全省资源库内容时，仍按既有权限规则过滤不可见资源。
- **数据规则**：
  - **取数规则**：权限状态从用户角色与模块权限配置读取。
  - **数据筛选规则**：仅当用户具备全省资源库查看权限时，返回新增模块所需数据。
  - **数据排序规则**：权限判断不改变已有模块的数据排序。

## 三、全省资源库搜索升级

### 3.1 搜索模式 Tab

- **展示规则**：
  - 新版搜索模块仅在“成品稿件”资源类型下展示。
  - 成品稿件搜索模块顶部提供“精准检索”和“语义检索”两个模式 Tab。
  - 默认选中“精准检索”。
  - 模式 Tab 采用醒目的顶部横向样式，便于与普通筛选项区分。
- **交互规则**：
  - 切换搜索模式时保留资源类型、渠道和发布时间条件。
  - 精准检索和语义检索输入方式不同，切换后展示对应输入区。
  - 点击“收起”可折叠高级条件，仅保留模式 Tab 和核心输入框。
  - 切换到成品图片、成品视频、成品音频时隐藏新版搜索模块，展示旧版搜索筛选模块。
- **数据规则**：
  - **取数规则**：读取当前搜索模式、输入内容和筛选条件。
  - **数据筛选规则**：搜索模式为单选条件，仅作用于成品稿件。
  - **数据排序规则**：精准检索默认按发布时间倒序；语义检索默认按相关度倒序。

### 3.2 精准检索

- **展示规则**：
  - 精准检索仅对成品稿件生效，提供字段下拉、关键词输入、关键词匹配方式和追加条件入口。
  - 字段下拉默认值为“稿件标题”，可按资源类型显示对应字段选项。
  - 关键词输入支持多个关键词用空格分隔。
  - 匹配方式默认“包含全部关键词”。
  - 追加条件入口使用“+”按钮；新增条件行左侧展示“或”连接标识，并提供删除入口。
- **交互规则**：
  - 点击“+”可增加一组检索条件，多组检索条件之间执行逻辑或运算。
  - 点击新增条件行的删除入口，可移除该组或条件。
  - 点击“重置条件”清空关键词、删除新增条件行、恢复默认字段和默认筛选项。
  - 点击“搜索”按当前条件刷新结果。
- **数据规则**：
  - **取数规则**：读取每一组检索条件的字段、关键词和匹配方式。
  - **数据筛选规则**：单组内多关键词按匹配方式计算；多组检索条件之间取并集，再与资源类型、渠道、地域、发布时间、稿件类型和检索设置取交集。
  - **数据排序规则**：默认按发布时间倒序；用户切换排序时按所选排序执行。

### 3.3 语义检索

- **展示规则**：
  - 语义检索仅对成品稿件生效，提供大文本输入框，用于输入自然语言检索意图。
  - 输入框 placeholder 引导用户描述要查找的报道或素材。
  - 下方复用发布时间、稿件类型、检索设置等高级条件。
- **交互规则**：
  - 点击搜索后，根据自然语言意图返回匹配资源。
  - 清空输入框后再次搜索时提示用户输入检索内容。
  - 语义检索结果需展示相关度排序效果。
- **数据规则**：
  - **取数规则**：读取用户输入的自然语言描述，并结合资源标题、摘要、标签、结构化信息进行语义匹配。
  - **数据筛选规则**：语义结果仍受资源类型、渠道、发布时间、地域、稿件类型等条件约束。
  - **数据排序规则**：默认按语义相关度倒序；相关度相同时按发布时间倒序。

### 3.4 发布时间筛选

- **展示规则**：
  - 发布时间包含全部、单日期、时间段、特殊日期四个选项。
  - 默认选中全部。
- **交互规则**：
  - 选择单日期时展示单个日期选择控件。
  - 选择时间段时展示开始日期和结束日期。
  - 选择特殊日期时展示常用日期范围选项。
- **数据规则**：
  - **取数规则**：读取资源发布时间。
  - **数据筛选规则**：按所选时间条件筛选资源；全部不限制时间。
  - **数据排序规则**：不改变当前排序方式。

### 3.5 稿件类型与检索设置

- **展示规则**：
  - 稿件类型包含新闻稿、图集稿、链接稿、视频稿、直播稿。
  - 检索设置包含忽略“无标题”稿件、仅查询原创稿。
  - 稿件类型与检索设置仅在成品稿件新版搜索模块中展示。
- **交互规则**：
  - 多个稿件类型可多选，选中后立即参与筛选。
  - 勾选检索设置后，点击搜索时生效。
- **数据规则**：
  - **取数规则**：读取资源内容类型、标题状态和原创标识。
  - **数据筛选规则**：稿件类型多选取并集；检索设置与其他条件取交集。
  - **数据排序规则**：不改变当前排序方式。

### 3.6 图片、视频、音频旧版搜索筛选

- **展示规则**：
  - 成品图片、成品视频、成品音频不展示新版精准检索和语义检索模块。
  - 旧版搜索筛选模块展示开始日期、结束日期、关键词搜索框和搜索按钮。
  - 搜索筛选模块下方展示批量选择栏，包含全选、已选数量和清除入口。
- **交互规则**：
  - 点击搜索按钮按日期范围和关键词刷新当前资源类型结果。
  - 勾选全选时选中当前页全部资源；取消全选时取消当前页全部选择。
  - 点击清除时取消当前页已选资源，并将已选数量恢复为 0。
- **数据规则**：
  - **取数规则**：读取当前资源类型、开始日期、结束日期、关键词和当前页选择状态。
  - **数据筛选规则**：日期范围、关键词、地域和客户端渠道取交集；当前资源类型仅取成品图片、成品视频或成品音频。
  - **数据排序规则**：默认按发布时间倒序，保持旧版资源列表排序口径。

## 四、全省资源库结果区

### 4.1 左侧地域筛选

- **展示规则**：
  - 左侧地域筛选按省本级和地市分组展示。
  - 地市分组可展开区县和市本级。
  - 地域节点不展示数量。
  - 已选地域展示重置入口。
- **交互规则**：
  - 勾选地市时，默认选中该地市下全部子项。
  - 勾选区县时，仅筛选该区县内容。
  - 重置入口可一次性恢复地域筛选默认状态。
- **数据规则**：
  - **取数规则**：根据资源发布站点对应的地域属性读取省、市、区县层级。
  - **数据筛选规则**：多地域取并集，与资源类型、渠道和搜索条件取交集。
  - **数据排序规则**：不影响结果排序。

### 4.2 成品稿件结果

- **展示规则**：
  - 成品稿件以表格展示，包含选择框、序号、稿件标题、来源、稿件类型、字数、发布站点、发布时间、操作。
  - 标题可点击进入详情。
  - 列表上方展示批量取稿按钮和当前结果数量。
- **交互规则**：
  - 支持单选、全选和批量取稿。
  - 点击单行“取稿”弹出取稿渠道选择。
  - 切换筛选条件后清空当前勾选状态。
- **数据规则**：
  - **取数规则**：读取全省资源库稿件标题、来源、类型、字数、发布站点、发布时间和可取稿状态。
  - **数据筛选规则**：按资源类型、渠道、地域、发布时间、稿件类型、关键词和检索设置取交集。
  - **数据排序规则**：默认按发布时间倒序；语义检索模式按相关度倒序。

### 4.3 成品图片、成品视频、成品音频结果

- **展示规则**：
  - 成品图片和成品视频以卡片网格展示；成品音频以列表展示。
  - 成品图片卡片展示选择框、封面、标题、来源和发布时间。
  - 成品视频卡片展示选择框、封面、播放标识、时长、标题、来源和发布时间。
  - 成品音频列表展示选择框、标题、来源、时长、格式、发布时间和操作。
  - 结果区上方使用旧版日期范围、关键词和批量选择筛选模块。
- **交互规则**：
  - 点击卡片或标题进入资源详情。
  - 图片、视频支持添加到个人素材库。
  - 音频支持播放预览和下载。
- **数据规则**：
  - **取数规则**：读取对应资源类型的标题、封面或预览、来源、发布时间、时长、格式等业务字段。
  - **数据筛选规则**：按资源类型、渠道、地域、发布时间和关键词取交集。
  - **数据排序规则**：默认按发布时间倒序；语义检索模式按相关度倒序。

### 4.4 分页

- **展示规则**：
  - 结果区底部展示总数、每页条数、页码、上一页、下一页和跳转页码。
- **交互规则**：
  - 点击页码切换页面。
  - 每页条数切换后回到第一页。
  - 页码越界时提示并保持当前页。
- **数据规则**：
  - **取数规则**：按当前筛选条件和页码截取结果。
  - **数据筛选规则**：分页不改变筛选条件。
  - **数据排序规则**：分页保持当前排序方式。

## 五、全局交互规则

- 所有可点击控件具备 hover、focus 和 active 状态。
- 搜索、筛选、资源类型切换、渠道切换均需提供即时视觉反馈。
- 无结果时展示空状态，提示用户调整关键词、时间或地域筛选。
- 统计图、推荐素材、资源列表中的数据仅展示当前用户权限范围内可见内容。
- 全省资源库页面视觉按政务后台风格收敛，使用白底浅边框、紧凑表单、低阴影和明确功能色，避免装饰性渐变和过重视觉元素。
- PRD 面板与原型页面平级展示，支持打开、收起、下载和查看更新记录。
