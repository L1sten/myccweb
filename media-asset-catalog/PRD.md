# 智能媒资编目与全省资源库稿件详情 PRD

**版本**：v1.6
**日期**：2026-06-25
**适用范围**：智能媒资系统素材库、全省资源库、素材详情、编目管理

---

## 1. 产品概述

### 1.1 背景

智能媒资系统已具备素材库管理、结构化分析、搜索、共享审批等基础能力。本期聚焦素材库文件夹上传、全省资源库稿件查看、素材编目和编目模板管理，提升稿件浏览、素材检索和元数据治理效率。

### 1.2 目标

- 全省资源库成品稿件列表新增不同稿件类型筛选，支持按图集稿件、视频稿件、视频专辑稿件、音频专辑稿件快速定位稿件。
- 个人素材库、部门素材库、公共素材库的上传按钮新增上传文件夹能力。
- 图集稿、视频稿、视频专辑稿、音频专辑稿从全省资源库进入详情页查看。
- 素材详情支持按模板填写编目信息，并保存草稿或完成编目。
- 编目管理作为独立模块，支持管理员维护通用编目模板和字段配置。

### 1.3 用户角色

- 采编人员：查看稿件详情、查看素材详情、补充编目信息、完成编目。
- 素材管理员：维护素材库、检查字段质量、管理模板。
- 审核人员：在审批中心审核共享入库申请及其编目信息。
- 系统管理员：配置素材类型、模板权限和字段规则。

### 1.4 使用场景

- 用户在全省资源库按稿件类型筛选成品稿件。
- 用户在个人素材库、部门素材库或公共素材库中通过上传按钮上传本地文件夹。
- 用户从全省资源库进入图集稿、视频稿、视频专辑稿、音频专辑稿详情。
- 采编人员在素材详情页按模板补齐编目信息。
- 管理员在编目管理中维护不同业务场景的编目模板。

### 1.5 本期范围

- 全省资源库成品稿件列表新增稿件类型筛选。
- 个人素材库、部门素材库、公共素材库上传按钮新增上传文件夹入口。
- 全省资源库图集稿、视频稿、视频专辑稿、音频专辑稿详情查看。
- 素材详情编目信息 Tab、保存草稿、完成编目、模板切换。
- 编目管理模块的模板列表、新建模板、编辑模板、复制、删除、设为默认。

### 1.6 非本期范围

- 不新增审批中心流程。
- 不新增智能分析模型配置。
- 不新增图像、视频、音频在线编辑能力。
- 不改变已有上传文件入口、上传限制和素材入库规则。
- 不改变已有素材下载、分享、取至个人的审批规则。

### 1.7 关键约束 / 待确认问题

- 已被素材使用的模板是否允许停用，待业务确认。
- 默认编目模板按系统配置执行。
- 全省资源库稿件类型、来源和发布站点选项以现有业务字典为准。

## 2. 全局交互规则

### 2.1 页面入口与导航

- 顶部主导航包含首页、搜索中心、资源中心、素材中心、审批中心、同步中心、标签中心、全省资源库、编目管理。
- 图集稿、视频稿、视频专辑稿、音频专辑稿的详情入口归属于全省资源库。
- 普通素材详情编目入口归属于素材详情。
- 编目模板维护入口归属于编目管理。

### 2.2 筛选与查询

- 本期新增稿件类型筛选；地域、时间、关键词搜索等筛选能力沿用现有全省资源库列表规则。
- 稿件类型筛选按图集稿件、视频稿件、视频专辑稿件、音频专辑稿件筛选。
- 多个筛选条件同时使用时取交集。
- 清空筛选后恢复当前模块默认列表。

### 2.3 分页与排序

- 列表默认按发布时间倒序排列。
- 同一发布时间的记录按系统默认稳定顺序排列。
- 支持按发布时间切换升序或降序。
- 空结果展示空状态，不保留上一轮查询结果。

### 2.4 按钮与操作反馈

- 无权限操作不展示入口；只读权限下展示详情但不展示编辑类按钮。
- 保存、删除、完成编目等操作执行中展示进行中状态，完成后刷新当前列表或详情信息。
- 删除等高风险操作需二次确认。
- 操作失败时展示失败原因，并保留用户已填写内容。

### 2.5 空状态与异常状态

- 列表无数据时展示空状态和重置筛选入口。
- 稿件详情媒体加载失败时展示占位状态，不影响稿件信息查看。
- 专辑无媒体条目时展示空状态，不展示上一条、下一条切换按钮。
- 专辑仅一条媒体时展示当前媒体，不提供上一条、下一条切换能力。

### 2.6 权限与可见性

- 用户仅可查看权限范围内的稿件、素材和模板。
- 编辑类入口根据权限隐藏或禁用；禁用时需展示不可操作原因。
- 管理类入口仅对具备编目模板管理权限的用户可见。
- 下载、分享、取至个人沿用现有系统权限规则。

## 3. 素材库上传文件夹

### 3.1 模块目标与边界

- 本期在个人素材库、部门素材库、公共素材库的上传按钮中新增上传文件夹能力。
- 上传文件夹用于批量上传本地文件夹内的素材，并保留文件夹内的嵌套层级。
- 本模块不改变已有上传文件入口、上传权限、上传限制、重名处理和素材入库规则。

### 3.2 入口规则

上传按钮：

- 展示规则：个人素材库、部门素材库、公共素材库中展示上传按钮；点击上传按钮后展示上传文件、上传文件夹两个选项。
- 交互规则：点击上传文件夹后调用系统文件夹选择窗口；用户取消选择时不创建上传任务。
- 权限规则：上传按钮和上传文件夹选项沿用当前素材库的上传权限；无上传权限时不展示上传入口。
- 数据规则：
  - 取数规则：取当前用户有上传权限的素材库范围。
  - 数据筛选规则：仅在个人素材库、部门素材库、公共素材库中提供上传文件夹入口。
  - 数据排序规则：不涉及排序。

### 3.3 上传文件夹规则

文件夹选择：

- 展示规则：系统文件夹选择窗口支持选择本地文件夹。
- 交互规则：用户选择文件夹后，系统读取该文件夹及其下级嵌套文件夹中的文件；空文件夹不生成上传文件。
- 数据规则：
  - 取数规则：取用户所选文件夹内符合上传规则的文件。
  - 数据筛选规则：不符合已有上传限制的文件按现有异常规则处理。
  - 数据排序规则：按文件夹内原始层级和文件顺序进入上传任务。

目录层级：

- 展示规则：上传任务中展示文件夹内的相对目录层级。
- 交互规则：上传完成后，素材在目标素材库中按相对目录层级归档。
- 数据规则：
  - 取数规则：取所选文件夹下各文件的相对路径。
  - 数据筛选规则：仅保留所选文件夹内部路径，不包含用户本机绝对路径。
  - 数据排序规则：按相对路径层级展示。

### 3.4 验收标准

- 个人素材库、部门素材库、公共素材库的上传按钮均提供上传文件和上传文件夹两个选项。
- 点击上传文件夹后打开系统文件夹选择窗口。
- 选择含嵌套子目录的文件夹后，上传任务包含子目录内符合上传规则的文件。
- 上传文件夹保留所选文件夹内的相对目录层级，不记录用户本机绝对路径。
- 用户取消文件夹选择时，不创建上传任务。
- 无上传权限的用户不展示上传入口。

## 4. 全省资源库稿件列表

### 4.1 模块目标与边界

- 本期在全省资源库成品稿件列表中新增稿件类型筛选能力。
- 既有地域筛选、发布时间筛选、关键词搜索、列表字段、分页和排序规则沿用线上全省资源库能力。
- 本模块不改变普通素材管理、编目模板维护、审批流程和上传流程。

### 4.2 页面结构

- 全省资源库稿件列表沿用现有列表结构。
- 筛选条新增稿件类型筛选控件，与现有时间筛选、地域筛选、关键词搜索位于同一筛选区域。
- 稿件列表支持混合展示不同类型稿件；每条记录展示自身稿件类型。
- 用户点击稿件标题进入对应类型的稿件详情页。

### 4.3 稿件类型筛选规则

稿件类型筛选：

- 展示规则：筛选条展示稿件类型筛选控件；默认展示全部稿件类型。
- 选项规则：选项包含全部稿件类型、图集稿件、视频稿件、视频专辑稿件、音频专辑稿件。
- 交互规则：用户选择稿件类型后刷新列表；清空后恢复全部稿件类型。
- 数据规则：
  - 取数规则：取当前用户权限范围内可查看的成品稿件。
  - 数据筛选规则：按稿件类型精确匹配；全部稿件类型不限制稿件类型。
  - 数据排序规则：筛选后沿用现有列表排序规则。
- 组合规则：稿件类型筛选与现有地域、发布时间、关键词搜索条件同时使用时取交集。
- 空状态：当前组合条件下无匹配稿件时，沿用现有空结果展示和重置筛选入口。

### 4.4 列表与详情入口联动

- 类型展示规则：列表中的类型字段展示当前稿件所属类型，至少包含图集稿件、视频稿件、视频专辑稿件、音频专辑稿件。
- 点击规则：点击稿件标题进入与类型匹配的详情页。
- 联动规则：
  - 图集稿件进入图集稿详情。
  - 视频稿件进入视频稿详情。
  - 视频专辑稿件进入视频专辑稿详情。
  - 音频专辑稿件进入音频专辑稿详情。
- 权限规则：无查看权限的稿件不展示在列表中；已展示稿件默认可点击进入详情。

### 4.5 验收标准

- 全省资源库稿件列表展示稿件类型筛选控件，默认展示全部稿件类型。
- 稿件类型筛选选项包含图集稿件、视频稿件、视频专辑稿件、音频专辑稿件。
- 选择任一稿件类型后，列表仅展示匹配该类型的稿件。
- 清空稿件类型筛选后，列表恢复展示全部稿件类型。
- 稿件类型筛选与现有地域、发布时间、关键词搜索条件同时使用时取交集。
- 列表中不同类型稿件可点击进入对应详情页。

## 5. 图集详情

### 5.1 页面结构

- 图集稿顶部采用稿件阅读样式，依次展示阅读原文按钮、稿件标题、来源、字数和发布时间。
- 左侧内容区按主图预览、图片图说、图片列表、正文内容顺序展示。
- 主图预览展示当前图片、当前序号和左右切换按钮。
- 图片图说展示在主图预览下方、图片列表上方，随当前图片切换变化。
- 图片列表展示图集内图片缩略图，当前图片在列表中高亮。
- 图片列表下方展示正文内容，正文内容为图集稿的稿件正文，切换图片时不变化。
- 右侧展示稿件级智能标签、智能关键词和智能摘要。

### 5.2 字段规则

稿件标题：

- 展示规则：在图集稿顶部展示图集总标题，只读展示，空值展示为空。
- 交互规则：不支持在详情页直接编辑；切换图片时不变化。
- 数据规则：
  - 取数规则：取当前图集稿件的标题。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：不涉及排序。

来源：

- 展示规则：在图集稿顶部展示稿件来源，只读展示，空值展示为空。
- 交互规则：切换图片时不变化。
- 数据规则：
  - 取数规则：取当前图集稿件的来源信息。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：不涉及排序。

发布时间：

- 展示规则：在图集稿顶部展示发布时间，精确到秒，空值展示为空。
- 交互规则：切换图片时不变化。
- 数据规则：
  - 取数规则：取当前图集稿件的发布时间。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：不涉及排序。

当前图片：

- 展示规则：主图预览展示当前图片和当前序号；无图片时展示空状态。
- 交互规则：点击左右切换按钮、图片序号点或图片列表缩略图后，同步更新主图、当前序号、图说和缩略图高亮。
- 数据规则：
  - 取数规则：取当前图集内被选中的图片。
  - 数据筛选规则：仅展示归属于当前图集稿的图片。
  - 数据排序规则：按图集内图片原始顺序展示。

图片图说：

- 展示规则：展示在主图预览下方、图片列表上方，只读展示；空值时不展示图说内容。
- 交互规则：切换当前图片时，图说同步更新为当前图片对应图说。
- 数据规则：
  - 取数规则：取当前图集内被选中图片的图说内容。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：与当前图片顺序保持一致。

正文内容：

- 展示规则：展示在图片列表下方，按稿件正文内容只读展示，不额外展示字段标题；正文为空时不展示。
- 交互规则：不支持在详情页直接编辑；切换图片时正文内容不变化。
- 数据规则：
  - 取数规则：取当前图集稿件的正文内容，保留原稿文本段落顺序。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：不涉及排序。

智能标签、智能关键词、智能摘要：

- 展示规则：展示在图集稿右侧智能信息区；智能标签以标签块展示，智能关键词以井号关键词展示，智能摘要以正文段落展示；无识别结果时不展示对应内容。
- 交互规则：智能摘要支持重新生成入口；切换图片时右侧稿件级智能信息不变化。
- 数据规则：
  - 取数规则：取当前图集稿的智能分析结果。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：标签和关键词按系统推荐顺序展示。

### 5.3 操作规则

阅读原文：

- 默认状态：当前稿件存在原文来源时，在图集稿顶部展示。
- 启用状态：存在可阅读原文时可点击。
- 禁用状态：原文来源缺失时不展示。
- 点击反馈：点击后打开原文阅读入口。
- 结果变化：不改变当前详情页数据。

图片切换：

- 默认状态：图集包含图片时展示图片列表；图集包含多张图片时展示左右切换按钮。
- 启用状态：存在多张图片时可切换上一张或下一张。
- 禁用状态：图集仅一张图片时不提供上一张、下一张切换能力。
- 点击反馈：切换后更新主图、序号点、当前序号、图片图说和图片列表高亮。
- 结果变化：右侧稿件级智能信息和正文内容不随图片切换变化。

### 5.4 验收标准

- 图集详情顶部展示阅读原文按钮、稿件标题、来源、字数和发布时间。
- 图集详情左侧按主图预览、图片图说、图片列表、正文内容顺序展示。
- 图片图说位于主图预览下方、图片列表上方。
- 图片列表位于正文内容上方。
- 点击缩略图、序号点或左右切换按钮可以切换主图。
- 图片切换后，同步更新当前序号、图片图说和缩略图高亮。
- 图片切换后，正文内容和右侧稿件级智能信息保持不变。
- 图集详情右侧展示智能标签、智能关键词和智能摘要。

## 6. 视频稿详情

### 6.1 页面结构

- 视频稿顶部采用稿件阅读样式，依次展示阅读原文按钮、稿件标题、来源、字数和发布时间。
- 主体区域展示当前视频播放器。
- 视频稿详情不展示右侧智能标签、智能关键词、智能摘要区域。

### 6.2 字段规则

稿件标题：

- 展示规则：在视频稿顶部展示稿件标题，只读展示，空值展示为空。
- 交互规则：不支持在详情页直接编辑。
- 数据规则：
  - 取数规则：取当前视频稿标题。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：不涉及排序。

来源：

- 展示规则：在视频稿顶部展示稿件来源，只读展示，空值展示为空。
- 交互规则：不支持在详情页直接编辑。
- 数据规则：
  - 取数规则：取当前视频稿来源。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：不涉及排序。

发布时间：

- 展示规则：在视频稿顶部展示发布时间，精确到秒，空值展示为空。

- 新建模板和编辑模板均以弹窗方式完成。
- 新建模板默认生成 2 个可编辑字段。
- 编辑模板可新增字段、删除字段并更新字段顺序。
- 默认模板、已被素材使用的模板不可直接删除。
- 保存模板失败时保留当前编辑内容。

## 11. 权限规则

### 11.1 编目权限规则

| 权限点 | 可见性与操作影响 |
|---|---|
| 查看编目信息 | 有权限时可查看素材或稿件的编目信息；无权限时不展示编目信息入口 |
| 编辑编目 | 有权限时可编辑编目字段、保存草稿、完成编目；无权限时编目信息只读 |
| 编目模板管理 | 有权限时展示编目管理和模板管理入口；无权限时不展示管理入口 |
| 审核编目 | 有权限时可审核带编目信息的共享申请 |

## 12. 验收标准

- 个人素材库、部门素材库、公共素材库的上传按钮均提供上传文件和上传文件夹两个选项。
- 点击上传文件夹后打开系统文件夹选择窗口。
- 上传文件夹支持嵌套子目录，并保留所选文件夹内的相对目录层级。
- 全省资源库稿件列表新增稿件类型筛选，默认展示全部稿件类型。
- 稿件类型筛选选项包含图集稿件、视频稿件、视频专辑稿件、音频专辑稿件。
- 选择任一稿件类型后，列表仅展示匹配该类型的稿件；清空后恢复全部类型。
- 点击稿件标题可进入图集稿、视频稿、视频专辑稿、音频专辑稿详情。
- 稿件类型筛选与现有地域、发布时间、关键词搜索条件同时作用于稿件列表。
- 素材详情编目信息展示模板选择、自动保存草稿、保存草稿、完成编目和模板管理。
- 必填字段缺失时允许保存草稿，不允许完成编目。
- 图集详情左侧展示主图、图片图说、图片列表和正文内容。
- 图集详情图片切换后更新主图、当前序号、图片图说和缩略图高亮。
- 图集详情图片切换后正文内容和右侧稿件级智能信息保持不变。
- 图集详情右侧展示智能标签、智能关键词和智能摘要。
- 视频稿详情展示顶部稿件信息和视频播放器，不展示右侧智能信息区域。
- 视频专辑稿详情支持上一条、下一条按顺序切换视频，首尾不循环。
- 音频专辑稿详情支持上一条、下一条按顺序切换音频，首尾不循环。
- 视频专辑稿和音频专辑稿在无条目、单条条目、多条条目下均有明确展示状态。
- 编目管理支持模板列表、新建模板、编辑模板、复制模板、删除模板和设为默认。
- 新建模板和编辑模板均使用弹窗。
- 保存模板时校验模板名称、适用范围、字段名和字段类型。
