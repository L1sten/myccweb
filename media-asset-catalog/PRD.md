# 智能媒资素材上传与编目优化 PRD

**版本**：v1.4
**日期**：2026-06-11
**适用范围**：智能媒资系统素材中心、素材详情、编目管理

---

## 1. 产品目标

智能媒资系统在现有素材上传、素材库管理、结构化分析、搜索、共享审批能力基础上，补齐文件夹上传、素材编目、专辑详情和编目模板管理能力，提升素材入库后的可检索、可复用和可治理水平。

### 1.1 核心目标

- 素材中心支持上传文件和上传文件夹，文件夹上传保留嵌套目录结构。
- 素材上传后可进入编目流程，补齐标题、标签、分类、版权、关联信息等检索字段。
- 素材详情支持按模板填写编目信息，并保存草稿或完成编目。
- 图集、视频专辑、音频专辑支持在详情页切换专辑内媒体条目。
- 编目管理作为独立模块，支持管理员维护编目模板和字段配置。

### 1.2 用户角色

- 采编人员：上传素材、查看素材详情、补充编目信息、完成编目。
- 素材管理员：维护素材库、批量编目、检查字段质量、管理模板。
- 审核人员：在审批中心审核共享入库申请及其编目信息。
- 系统管理员：配置上传限制、素材类型、模板权限、自动结构化开关和字段规则。

## 3. 上传能力

### 3.1 上传入口

**展示规则**

- 上传按钮点击后展示两个选项：上传文件、上传文件夹。
- 上传文件夹入口直接调用系统文件夹选择窗口。

**交互规则**

- 点击上传文件直接调用系统文件选择窗口。
- 点击上传文件夹直接调用系统文件夹选择窗口。
- 文件夹选择支持嵌套子目录。
- 用户取消系统选择窗口时，不创建上传任务。
- 选择完成后打开上传任务抽屉。

**数据规则**

- 文件夹上传保留相对目录路径。
- 目标路径下按文件夹层级创建目录。
- 文件名、目录名、上传人、目标库、目标路径、文件数量、上传批次号进入上传任务记录。
- 压缩包文件通过上传文件入口处理。

### 3.2 上传任务抽屉

**展示规则**

- 展示当前批次的文件数量、预计体积、异常文件数、待编目数量。
- 展示目标目录、重名处理、默认版权状态。
- 文件夹上传展示目录结构。
- 抽屉底部提供上传任务和进入编目工作台。

**交互规则**

- 用户可调整目标目录。
- 用户可设置重名处理策略：跳过、保留两份并自动改名、覆盖旧文件。
- 上传完成后可进入当前批次编目工作台。

## 4. 素材详情编目

### 4.1 页面结构

- 详情页采用左侧预览、右侧信息面板布局。
- 左侧展示素材预览、大小、格式、尺寸或时长、原创状态。
- 顶部操作区包含返回列表、下载、分享、取至个人、以图搜图、图搜视频。
- 右侧 Tab 包含基本信息、编目信息、结构化信息、共享记录、敏感信息识别、下载记录。

### 4.2 编目信息

**展示规则**

- 编目信息 Tab 展示编目模板选择、自动保存草稿、保存草稿、完成编目、模板管理。
- 字段按当前模板配置动态展示。
- 模板可配置字段分组、字段名称、字段类型、必填状态、占位文字、默认值和排序。
- 人物库、新闻图片、非遗文化、音视频节目等均作为可配置模板示例，不作为固定字段范围。

**交互规则**

- 用户选择编目模板后，右侧表单按模板字段生成。
- 保存草稿仅保存当前填写内容。
- 完成编目校验当前模板必填字段。
- 模板管理入口跳转至编目管理模块。

### 4.3 通用模板字段配置

| 配置项 | 说明 |
|---|---|
| 基础字段 | 内部标识号、资源名称等系统通用字段 |
| 业务字段 | 按模板配置人物、地点、事件、栏目、版权、标签、来源、领域等字段 |
| 关联字段 | 按模板配置相关人物、相关地点、相关新闻、关联素材等字段 |
| 字段分组 | 支持按基础信息、业务信息、关联信息等分组展示 |
| 字段类型 | 支持文本、数字、下拉、日期、多行文本等类型 |
| 字段控制 | 支持必填、占位文字、默认值和排序配置 |

### 4.4 数据规则

- 内部标识号由系统自动生成。
- 资源名称默认取文件名，用户可修改。
- 系统自动字段包含格式、大小、尺寸、时长、上传人、上传时间。
- 版权状态缺失时允许保存草稿，不允许完成编目。

## 5. 图集详情

### 5.1 页面结构

- 顶部操作区包含返回列表、下载、分享、以图搜图、阅读原文。
- 左侧内容区按主图预览、图片图说、图片列表、正文内容顺序展示。
- 主图预览展示当前图片、当前序号和左右切换按钮。
- 图片图说展示在主图预览下方、图片列表上方，随当前图片切换变化。
- 图片列表展示图集内图片缩略图，当前图片在列表中高亮。
- 图片列表下方展示正文内容，正文内容为图集稿的稿件正文，切换图片时不变化。
- 切换当前图片时，同步更新主图预览、图片图说、当前序号、图片列表高亮状态，以及右侧当前图片的格式、尺寸、大小。
- 右侧仅展示基本信息，不展示编目信息、结构化信息、共享记录、敏感信息识别、下载记录等其它字段页。

### 5.2 字段规则

稿件标题：

- 展示规则：在右侧基本信息中展示图集总标题，只读展示，空值展示为空。
- 交互规则：不支持在详情页直接编辑；切换图片时不变化。
- 数据规则：
  - 取数规则：取当前图集稿件的标题。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：不涉及排序。

当前图片信息：

- 展示规则：右侧基本信息展示当前图片的格式、尺寸、大小；左侧主图预览展示当前图片序号。
- 交互规则：点击左右切换按钮或图片列表缩略图后，当前图片信息随之更新。
- 数据规则：
  - 取数规则：取当前图集内被选中图片的媒体属性。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：按图集内图片原始顺序展示。

图片图说：

- 展示规则：展示在主图预览下方、图片列表上方，只读展示；空值时不展示图说内容。
- 交互规则：切换当前图片时，图说同步更新为当前图片对应图说。
- 数据规则：
  - 取数规则：取当前图集内被选中图片的图说内容。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：与当前图片顺序保持一致。

来源：

- 展示规则：在右侧基本信息中展示稿件来源，只读展示，空值展示为空。
- 交互规则：切换图片时不变化。
- 数据规则：
  - 取数规则：取当前图集稿件的来源信息；来源与发布媒体默认同源，若业务来源与实际发布平台不同，来源展示素材归集来源。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：不涉及排序。

素材数量：

- 展示规则：在右侧基本信息中展示当前图集关联的图片数量。
- 交互规则：图片切换不改变素材数量。
- 数据规则：
  - 取数规则：统计当前图集内关联图片素材数量。
  - 数据筛选规则：仅统计归属于当前图集的图片素材。
  - 数据排序规则：不涉及排序。

发布时间：

- 展示规则：在右侧基本信息中展示发布时间，精确到秒，空值展示为空。
- 交互规则：切换图片时不变化。
- 数据规则：
  - 取数规则：取当前图集稿件的发布时间。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：不涉及排序。

图片列表：

- 展示规则：展示图集内图片缩略图和图片序号点，默认选中第一张图片并高亮；仅一张图片时展示单张缩略图并高亮；图片数量较多时支持在图片列表区域内横向浏览；无图片时主图预览展示空状态，当前序号不展示，右侧格式、尺寸、大小展示为空；图片加载失败时展示图片占位并保留原排序位置。
- 交互规则：点击缩略图或图片序号点切换当前图片；多张图片时点击左右切换按钮在图片列表内切换上一张或下一张，首尾支持循环切换；横向浏览缩略图列表本身不切换主图；切换图片后当前缩略图保持可见；仅一张图片时不提供上一张、下一张切换能力。
- 数据规则：
  - 取数规则：取当前图集稿件关联的图片素材。
  - 数据筛选规则：仅展示归属于当前图集的图片素材。
  - 数据排序规则：按图集内图片原始顺序展示。

正文内容：

- 展示规则：展示在图片列表下方，按稿件正文内容只读展示，不额外展示字段标题；正文为空时该区域不展示内容；正文较长时按页面滚动查看，不截断；多段正文按原稿段落顺序展示。
- 交互规则：不支持在详情页直接编辑；切换图片时正文内容不变化。
- 数据规则：
  - 取数规则：取当前图集稿件的正文内容，保留原稿文本段落顺序。
  - 数据筛选规则：不涉及筛选。
  - 数据排序规则：不涉及排序。

### 5.3 操作规则

阅读原文：

- 默认状态：当前稿件存在原文来源时，在顶部操作区展示。
- 点击反馈：点击后打开原文阅读入口。
- 异常状态：原文来源缺失时不展示该按钮。

顶部通用操作：

- 默认状态：展示返回列表、下载、分享、以图搜图、阅读原文。
- 启用状态：返回列表默认可点击；下载要求具备下载素材权限；分享要求具备分享素材权限；以图搜图要求具备以图搜图权限；阅读原文要求当前稿件存在原文来源；同时当前稿件状态需允许对应操作。
- 禁用状态：当前稿件缺少可下载文件、不可分享、不支持以图搜图或缺少原文来源时，对应按钮置为不可用或不展示；不额外展示禁用原因。
- 权限规则：无权限用户不展示对应按钮。
- 点击反馈：返回列表回到素材中心列表；下载当前图集稿件包含的全部图片；分享进入分享流程；以图搜图以当前主图作为检索对象进入搜索结果；阅读原文打开原文阅读入口。
- 结果变化：下载和分享完成后保留当前详情页状态；返回列表、以图搜图和阅读原文进入对应结果页面。

图片切换：

- 默认状态：图集包含图片时展示图片列表；图集包含多张图片时展示左右切换按钮。
- 点击反馈：切换后更新主图、序号点、当前序号、图片列表高亮和右侧当前图片属性。
- 异常状态：图集无图片时展示空状态；图集仅包含一张图片时，不提供上一张、下一张切换能力。

### 5.4 验收标准

- 图集详情左侧按主图预览、图片图说、图片列表、正文内容顺序展示。
- 图片图说位于主图预览下方、图片列表上方。
- 图片列表位于正文内容上方。
- 点击缩略图可以切换主图，并同步更新当前序号、缩略图高亮和右侧当前图片属性。
- 点击图片序号点可以切换主图，并同步更新缩略图高亮和右侧当前图片属性。
- 点击左右切换按钮可以循环切换图片。
- 图片切换后，图片图说同步更新为当前图片图说。
- 正文内容在图片切换时保持不变。
- 右侧仅展示基本信息字段。

## 8. 编目管理

### 8.1 模板列表

**展示规则**

- 编目管理作为顶部独立模块进入。
- 模板列表展示模板名称、描述、字段数量、创建时间、更新时间、默认标识和操作按钮。
- 模板列表支持展示适用范围，包括素材类型、素材库范围或业务场景。
- 操作按钮包含设为默认、编辑、复制、删除。
- 默认模板展示默认标识。
- 人物库模板仅为内置示例模板之一，管理员可按业务需要新增任意编目模板。

### 8.2 新建与编辑模板

**展示规则**

- 新建模板和编辑模板均使用弹窗。
- 弹窗包含模板名称、模板描述、适用范围、字段配置、取消、保存模板。
- 字段配置包含字段名、类型、分组、占位文字、必填、删除操作。
- 字段类型支持文本、数字、下拉、日期、多行文本。

**交互规则**

- 点击新建模板打开弹窗，默认生成 2 个可编辑字段。
- 点击编辑打开弹窗并加载当前模板字段。
- 点击添加字段在弹窗内新增字段。
- 点击删除字段移除当前字段并重排序号。
- 保存模板校验模板名称、字段名和字段类型。

### 8.3 数据规则

- 模板支持按素材类型、素材库、部门或业务场景配置适用范围。
- 模板字段保存字段 ID、字段名、类型、分组、占位文字、必填、排序。
- 同一素材库范围内仅允许一个默认模板。
- 已被素材使用的模板不允许直接删除。

## 10. 验收标准

- 上传按钮下仅展示上传文件、上传文件夹两个选项。
- 点击上传文件夹直接弹出系统文件夹选择窗口。
- 文件夹上传支持嵌套目录并保留相对路径。
- 素材列表展示图集、视频专辑和音频专辑卡片。
- 图集详情右侧仅展示基本信息。
- 视频专辑详情右侧仅展示基本信息。
- 音频专辑详情右侧仅展示基本信息。
- 图集详情支持左右切换当前图片。
- 视频专辑详情支持顶部左右切换当前视频。
- 音频专辑详情支持顶部左右切换当前音频。
- 素材详情编目信息展示模板选择、保存草稿、完成编目和模板管理。
- 素材详情支持选择编目模板、保存草稿、完成编目和进入模板管理。
- 编目管理支持新建模板弹窗和编辑模板弹窗。
- 新建模板默认生成 2 个可编辑字段。
- 编辑模板可新增字段、删除字段并更新字段序号。
- PRD 独立 Markdown 与 HTML 内嵌 PRD 内容保持一致。
