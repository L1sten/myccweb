# 智能媒资素材上传与编目优化 PRD

**版本**：v1.4
**日期**：2026-06-11
**适用范围**：智能媒资系统素材中心、素材详情、编目管理

---

## 1. 产品目标

智能媒资系统在现有素材上传、素材库管理、结构化分析、搜索、共享审批能力基础上，补齐文件夹上传、素材编目、专辑详情和编目模板管理能力，提升素材入库后的可检索、可复用和可治理水平。

### 1.1 核心目标

- 素材中心支持上传文件和上传文件夹，文件夹上传保留嵌套目录结构。
- 素材上传后可进入编目流程，补齐标题、标签、分类、版权、关联信息等检索字段。
- 素材详情支持按模板填写编目信息，并保存草稿或完成编目。
- 视频专辑、音频专辑支持在详情页顶部切换专辑内音视频条目。
- 编目管理作为独立模块，支持管理员维护编目模板和字段配置。

### 1.2 用户角色

- 采编人员：上传素材、查看素材详情、补充编目信息、完成编目。
- 素材管理员：维护素材库、批量编目、检查字段质量、管理模板。
- 审核人员：在审批中心审核共享入库申请及其编目信息。
- 系统管理员：配置上传限制、素材类型、模板权限、自动结构化开关和字段规则。

## 2. 素材中心列表

### 2.1 展示规则

- 顶部主导航包含首页、搜索中心、资源中心、素材中心、审批中心、同步中心、智能标签库、编目管理等模块。
- 素材中心首屏保持线上素材管理页布局，包含类型 Tab、操作条、筛选搜索条、选中条、库位置入口和卡片网格。
- 类型 Tab 包含全部、视频、音频、图片、文件。
- 操作条包含上传、新建、下载、分享、取至个人、移动、删除、批量编目。
- 勾选素材后展示全选、已选数量和清除。
- 卡片视图支持文件夹、图片、视频、文件、视频专辑、音频专辑混合展示。
- 视频卡片展示播放标识；原创素材展示原创标签；专辑卡片展示专辑类型和条目数量。

### 2.2 交互规则

- 点击普通素材卡片进入素材详情编目页。
- 点击视频专辑卡片进入视频专辑详情页。
- 点击音频专辑卡片进入音频专辑详情页。
- 点击批量编目进入批量编目处理界面。

## 3. 上传能力

### 3.1 上传入口

**展示规则**

- 上传按钮点击后展示两个选项：上传文件、上传文件夹。
- 上传文件夹入口直接调用系统文件夹选择窗口。

**交互规则**

- 点击上传文件直接调用系统文件选择窗口。
- 点击上传文件夹直接调用系统文件夹选择窗口。
- 文件夹选择支持嵌套子目录。
- 用户取消系统选择窗口时，不创建上传任务。
- 选择完成后打开上传任务抽屉。

**数据规则**

- 文件夹上传保留相对目录路径。
- 目标路径下按文件夹层级创建目录。
- 文件名、目录名、上传人、目标库、目标路径、文件数量、上传批次号进入上传任务记录。
- 压缩包文件通过上传文件入口处理。

### 3.2 上传任务抽屉

**展示规则**

- 展示当前批次的文件数量、预计体积、异常文件数、待编目数量。
- 展示目标目录、重名处理、默认版权状态。
- 文件夹上传展示目录结构。
- 抽屉底部提供上传任务和进入编目工作台。

**交互规则**

- 用户可调整目标目录。
- 用户可设置重名处理策略：跳过、保留两份并自动改名、覆盖旧文件。
- 上传完成后可进入当前批次编目工作台。

## 4. 素材详情编目

### 4.1 页面结构

- 详情页采用左侧预览、右侧信息面板布局。
- 左侧展示素材预览、大小、格式、尺寸或时长、原创状态。
- 顶部操作区包含返回列表、下载、分享、取至个人、以图搜图、图搜视频。
- 右侧 Tab 包含基本信息、编目信息、结构化信息、共享记录、敏感信息识别、下载记录。

### 4.2 编目信息

**展示规则**

- 编目信息 Tab 展示编目模板选择、自动保存草稿、保存草稿、完成编目、模板管理。
- 字段按当前模板配置动态展示。
- 模板可配置字段分组、字段名称、字段类型、必填状态、占位文字、默认值和排序。
- 人物库、新闻图片、非遗文化、音视频节目等均作为可配置模板示例，不作为固定字段范围。

**交互规则**

- 用户选择编目模板后，右侧表单按模板字段生成。
- 保存草稿仅保存当前填写内容。
- 完成编目校验当前模板必填字段。
- 模板管理入口跳转至编目管理模块。

### 4.3 通用模板字段配置

| 配置项 | 说明 |
|---|---|
| 基础字段 | 内部标识号、资源名称等系统通用字段 |
| 业务字段 | 按模板配置人物、地点、事件、栏目、版权、标签、来源、领域等字段 |
| 关联字段 | 按模板配置相关人物、相关地点、相关新闻、关联素材等字段 |
| 字段分组 | 支持按基础信息、业务信息、关联信息等分组展示 |
| 字段类型 | 支持文本、数字、下拉、日期、多行文本等类型 |
| 字段控制 | 支持必填、占位文字、默认值和排序配置 |

### 4.4 数据规则

- 内部标识号由系统自动生成。
- 资源名称默认取文件名，用户可修改。
- 系统自动字段包含格式、大小、尺寸、时长、上传人、上传时间。
- 版权状态缺失时允许保存草稿，不允许完成编目。

## 5. 视频专辑详情

### 5.1 页面结构

- 顶部操作区包含返回列表、下载、分享、以图搜图、图搜视频。
- 左侧展示当前视频播放器、当前视频切换和稿件正文。
- 当前视频切换位于播放器下方、稿件正文上方。
- 切换当前视频时，同步更新上方播放器展示。
- 右侧采用素材详情统一标签页：基本信息、编目信息、结构化信息、共享记录、敏感信息识别、下载记录。
- 专辑信息归属于基本信息 Tab，默认选中基本信息。

### 5.2 顶部切换

**展示规则**

- 当前视频播放器展示当前序号、视频标题、格式、时长、大小。
- 当前视频切换区展示视频标题、格式、时长、大小。
- 左右按钮位于当前视频区域两侧。

**交互规则**

- 点击左侧按钮切换上一个视频。
- 点击右侧按钮切换下一个视频。
- 切换到首尾时循环切换。
- 切换后更新标题、格式、时长和大小。

### 5.3 字段

| 分组 | 字段 |
|---|---|
| 稿件信息 | 稿件ID、来源、列表标题 |
| 正文信息 | 稿件正文 |
| 共性字段 | 版权信息、标签、评论设置、点赞设置、签发后隐藏、相关新闻、备注 |
| 视频信息 | 视频类型、拍摄地点、电视台、拍摄时间、导演、主持人 |
| 当前视频 | 标题、格式、时长、大小 |

## 7. 编目管理

### 7.1 模板列表

**展示规则**

- 编目管理作为顶部独立模块进入。
- 模板列表展示模板名称、描述、字段数量、创建时间、更新时间、默认标识和操作按钮。
- 模板列表支持展示适用范围，包括素材类型、素材库范围或业务场景。
- 操作按钮包含设为默认、编辑、复制、删除。
- 默认模板展示默认标识。
- 人物库模板仅为内置示例模板之一，管理员可按业务需要新增任意编目模板。

### 7.2 新建与编辑模板

**展示规则**

- 新建模板和编辑模板均使用弹窗。
- 弹窗包含模板名称、模板描述、适用范围、字段配置、取消、保存模板。
- 字段配置包含字段名、类型、分组、占位文字、必填、删除操作。
- 字段类型支持文本、数字、下拉、日期、多行文本。

**交互规则**

- 点击新建模板打开弹窗，默认生成 2 个可编辑字段。
- 点击编辑打开弹窗并加载当前模板字段。
- 点击添加字段在弹窗内新增字段。
- 点击删除字段移除当前字段并重排序号。
- 保存模板校验模板名称、字段名和字段类型。

### 7.3 数据规则

- 模板支持按素材类型、素材库、部门或业务场景配置适用范围。
- 模板字段保存字段 ID、字段名、类型、分组、占位文字、必填、排序。
- 同一素材库范围内仅允许一个默认模板。
- 已被素材使用的模板不允许直接删除。

## 8. 权限与配置

| 权限点 | 说明 |
|---|---|
| 上传文件 | 使用现有上传文件能力 |
| 上传文件夹 | 使用系统文件夹选择并保留目录结构 |
| 查看素材详情 | 查看当前权限范围内素材详情 |
| 编辑编目 | 编辑素材编目字段 |
| 批量编目 | 对多条素材执行批量编目 |
| 编目模板管理 | 新建、编辑、复制、删除、设置默认模板 |
| 审核编目 | 审核带编目信息的共享申请 |

### 8.1 系统参数

- 单次上传最大文件数。
- 单文件最大体积。
- 文件夹最大层级。
- 文件路径最大长度。
- 默认重名策略。
- 默认版权状态。
- 默认编目模板。
- 内部标识号前缀和序号规则。
- 自动结构化开关默认值。

## 9. 验收标准

- 上传按钮下仅展示上传文件、上传文件夹两个选项。
- 点击上传文件夹直接弹出系统文件夹选择窗口。
- 文件夹上传支持嵌套目录并保留相对路径。
- 素材列表展示视频专辑和音频专辑卡片。
- 视频专辑详情支持顶部左右切换当前视频。
- 音频专辑详情支持顶部左右切换当前音频。
- 素材详情编目信息展示模板选择、保存草稿、完成编目和模板管理。
- 素材详情支持选择编目模板、保存草稿、完成编目和进入模板管理。
- 编目管理支持新建模板弹窗和编辑模板弹窗。
- 新建模板默认生成 2 个可编辑字段。
- 编辑模板可新增字段、删除字段并更新字段序号。
- PRD 独立 Markdown 与 HTML 内嵌 PRD 内容保持一致。
