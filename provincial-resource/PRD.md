# 全省资源库 PRD

**版本**: v1.0
**日期**: 2026-05-06
**作者**: 产品团队

---

## 1. 产品概述

### 1.1 背景

现有资源中心仅包含"网络资源库"，提供网络稿件、视频、图片、音频等素材类资源的检索与浏览。全省各级媒体单位产出的成品稿件（新闻、专题报道等）缺乏统一的汇聚与共享入口，市县融媒体中心的优质内容难以被省级媒体发现和选用。

### 1.2 产品定位

面向全省媒体从业人员的成品稿件共享平台，汇聚省本级、浙报集团及各市县融媒体中心已发布的成品类内容，支持按渠道、地域、时间、类型等多维度检索和浏览，促进优质内容在全省范围内的流通与复用。

### 1.3 用户角色

- **省级媒体编辑**：浏览、筛选市县优质稿件，择优选用或二次加工
- **市县融媒体中心编辑**：查看其他地区稿件，了解全省报道动态
- **平台管理员**：管理共享规则、审核共享内容

### 1.4 核心功能概览

在资源中心导航下拉菜单中新增"全省资源库"入口（与"网络资源库"平级），提供成品稿件的列表浏览、多条件筛选、地域筛选和稿件详情查看功能。

## 3. 功能需求

### 3.1 导航入口

**功能描述**：在顶部导航栏"资源中心"下拉菜单中新增"全省资源库"入口，与"网络资源库"平级，位于其右侧。

**展示规则**：
- 下拉菜单分为左右两个并列区域
- 左侧区域：网络资源库（含子菜单：网络稿件、网络视频、网络图片、网络音频）
- 右侧区域：全省资源库（仅成品类，无子菜单分类，直接进入列表页）
- 两个区域之间以竖线分隔

**交互规则**：
- 鼠标悬停"资源中心"时展开下拉菜单
- 点击"全省资源库"区域直接跳转至全省资源库列表页
- 当前页面为全省资源库时，对应区域高亮显示

### 3.2 顶部渠道 Tab

**功能描述**：页面顶部提供渠道一级分类 Tab，用于按发布渠道筛选稿件。

- **全部**：展示所有渠道稿件（默认选中）
- **客户端**：仅展示发布于客户端的稿件
- **网站**：仅展示发布于网站的稿件
- **数字报**：仅展示发布于数字报的稿件

**交互规则**：
- 点击 Tab 切换对应渠道，高亮当前选中
- 切换后列表自动刷新

**数据规则**：
- 取数规则：匹配稿件的"渠道"字段
- 数据筛选规则：单选，选中后仅显示对应渠道
- 数据排序规则：保持当前排序设置不变

### 3.3 左侧地域筛选

**页面结构**：左侧边栏采用二级可折叠结构。

**二级节点（默认展开省本级）**：
- 省本级
- 杭州市、湖州市、宁波市、温州市、嘉兴市、绍兴市、金华市、衢州市、舟山市、台州市、丽水市

**三级节点**：
- 省本级下：浙报集团
- 各地级市下：该市所辖区县（如杭州市下有西湖区、萧山区、桐庐县等）

**交互规则**：
- 点击二级节点展开/折叠三级子节点
- 每个节点前有复选框，支持多选
- 勾选二级节点时，该节点下所有三级节点全部选中
- 选中后自动刷新列表

**数据规则**：
- 取数规则：匹配稿件的"来源地域"字段
- 数据筛选规则：多选取并集（OR 逻辑），与其他筛选条件取交集

### 3.4 全省资源库列表页

**功能描述**：展示全省范围内已共享的成品稿件列表，支持多条件筛选和排序。

#### 筛选条件栏

- **日期范围**：两个日期输入框，默认最近一个月
- **稿件类型**：下拉选择（全部稿件/新闻/评论/专题/图集）
- **产品类型**：下拉选择（全部产品/客户端/网站/报纸/微信/微博）
- **排序**：下拉选择（时间/字数正序/字数倒序），默认按时间倒序
- **搜索**：关键词搜索，多关键词空格分隔取交集

#### 稿件列表

**列表表头**：序号、稿件标题、来源、发布产品、共享时间

**字段规则**：
- **稿件标题**：蓝色文字带链接图标，点击进入详情页
- **来源**：显示来源机构名称（如"平阳县融媒体中心"）
- **发布产品**：发布产品名称（如"爱平阳客户端"）
- **共享时间**：格式 YYYY/MM/DD HH:mm:ss

**分页规则**：默认每页 20 条

### 3.5 稿件详情页

**功能描述**：展示单篇稿件的完整内容。

**页面结构**：
- 左侧主区域：稿件标题 + 元信息（发布站点、渠道、发布时间）+ 正文内容
- 右侧辅助区：结构化信息面板（未分析时显示"暂无结构化信息"+"去分析"按钮）

**返回操作**：左上角返回按钮，返回列表页并保留筛选状态。进入详情页时左侧地域边栏自动隐藏。

---

## 4. 交互规范

- 所有筛选条件变更后自动刷新列表
- 无数据时显示"暂无数据"空状态提示
- 从列表页进入详情页时左侧地域边栏自动隐藏
- 从详情页返回时恢复边栏和之前的筛选状态

## 5. 数据模型

### 5.1 稿件实体

```text
Article {
  id: string [必填] 稿件唯一标识
  title: string [必填] 稿件标题
  content: string [必填] 正文内容
  source: string [必填] 来源机构
  sourceRegion: string [必填] 来源地域
  type: enum [必填] 稿件类型（新闻/评论/专题/图集）
  wordCount: number [必填] 正文字数
  publishProduct: string [必填] 发布产品名称
  productType: enum [必填] 产品类型
  publishSite: string [选填] 发布站点
  channel: enum [选填] 渠道（网站/数字报/客户端/微信/微博）
  publishTime: datetime [选填] 发布时间
  shareTime: datetime [必填] 共享时间
  structuredInfo: object [选填] 结构化分析结果
}
```
