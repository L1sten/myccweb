**版本**: v1.2
**日期**: 2026-05-07
**作者**: 产品团队

## 1. 产品概述

### 1.1 背景

现有资源中心仅包含"网络资源库"，提供网络稿件、视频、图片、音频等素材类资源的检索与浏览。全省各级媒体单位产出的成品稿件（新闻、专题报道等）缺乏统一的汇聚与共享入口，市县融媒体中心的优质内容难以被省级媒体发现和选用。

### 1.2 产品定位

面向全省媒体从业人员的成品内容共享平台，汇聚省本级、浙报集团及各市县融媒体中心已发布的成品类内容（稿件、图片、视频），支持按渠道、地域、时间等多维度检索和浏览，促进优质内容在全省范围内的流通与复用。

### 1.3 用户角色

- **省级媒体编辑**：浏览、筛选市县优质内容，择优选用或二次加工
- **市县融媒体中心编辑**：查看其他地区内容，了解全省报道动态
- **平台管理员**：管理共享规则、审核共享内容

### 1.4 核心功能概览

在资源中心导航下拉菜单中新增"全省资源库"入口（与"网络资源库"平级），提供三个子模块：成品稿件（列表视图）、成品图片（卡片网格视图）、成品视频（卡片网格视图），支持多条件筛选、地域侧栏筛选、批量取稿和分页功能。

## 3. 功能需求

### 3.1 导航入口

**功能描述**：在顶部导航栏"资源中心"下拉菜单中新增"全省资源库"入口，与"网络资源库"平级，位于其右侧。

**展示规则**：
- 下拉菜单分为左右两个并列区域
- 左侧区域：网络资源库，含子菜单项（网络稿件、网络视频、网络图片、网络音频），每项前带图标
- 右侧区域：全省资源库，含子菜单项（成品稿件、成品图片、成品视频），每项前带图标，文字高亮显示
- 两个区域之间以竖线分隔
- 当前页面所属子项无跳转，其余子项点击跳转对应页面

**交互规则**：
- 鼠标悬停"资源中心"时展开下拉菜单，移出后自动收起
- 当前页面为全省资源库任一子页面时，"资源中心"导航文字高亮显示
- 下拉菜单内全省资源库区域整体高亮（标题和文字使用品牌色）

### 3.2 顶部渠道 Tab

**功能描述**：页面顶部提供渠道一级分类 Tab，用于按发布渠道筛选内容。三个子页面共用相同的 Tab 结构。

- **全部**：展示所有渠道内容（默认选中）
- **客户端**：仅展示发布于客户端的内容
- **网站**：仅展示发布于网站的内容

**展示规则**：
- Tab 位于页面顶部，横向排列
- 当前选中项文字高亮并显示底部边线

**交互规则**：
- 点击 Tab 切换对应渠道，高亮当前选中
- 切换后列表或网格自动刷新

**数据规则**：
- **取数规则**：匹配内容的"渠道"字段
- **数据筛选规则**：单选，选中后仅显示对应渠道；"全部"不筛选
- **数据排序规则**：保持当前排序设置不变

### 3.3 左侧地域筛选

**功能描述**：左侧边栏按地域分组展示可折叠的地区列表，支持多选和"仅筛选此项"快捷操作。三个子页面共用相同的侧栏交互。

**展示规则**：
- 侧栏采用可折叠分组结构
- 分组节点包括：省本级、11 个地级市（杭州市、湖州市、宁波市、温州市、嘉兴市、绍兴市、金华市、衢州市、舟山市、台州市、丽水市）
- 每个分组节点前有复选框和展开/折叠箭头图标
- 展开后显示子节点列表，每个子节点前有复选框
- 子节点列表：
  - 省本级下：浙报集团
  - 各地级市下：市本级（固定为第一个子项）+ 该市所辖区县
- 子节点悬停时右侧显示漏斗图标
- 漏斗图标悬停时展示提示文字"仅筛选此项"
- 分组节点的展开/折叠按钮旁不显示漏斗图标

**交互规则**：
- 点击分组节点展开/折叠子节点
- 勾选分组节点时，该节点下所有子节点全部选中
- 勾选子节点时，仅选中或取消该节点，同组其他已选项保持不变
- 点击漏斗图标触发"仅筛选此项"：勾选当前子项，同组其他子项取消选中
- 选中后自动刷新列表

**数据规则**：
- **取数规则**：匹配内容的"来源地域"字段，关联到对应地级市
- **数据筛选规则**：多选取并集（OR 逻辑），与其他筛选条件取交集
- **数据排序规则**：不影响排序

### 3.4 成品稿件列表页

**功能描述**：展示全省范围内已共享的成品稿件列表，支持多条件筛选、排序、批量取稿和分页。

#### 筛选条件栏

- **日期范围**
  - 展示规则：两个日期输入框
  - 交互规则：选择日期后自动刷新列表
  - 数据规则：默认最近一个月；匹配稿件的"发布时间"字段，取日期范围区间

- **稿件类型**
  - 展示规则：下拉选择
  - 交互规则：选中后自动刷新列表
  - 数据规则：选项为全部稿件、新闻、评论、专题、图集；匹配稿件的"稿件类型"字段，精确匹配

- **排序**
  - 展示规则：下拉选择
  - 交互规则：选中后自动刷新列表
  - 数据规则：选项为时间倒序、字数正序、字数倒序；默认按时间倒序

- **搜索**
  - 展示规则：搜索输入框
  - 交互规则：输入关键词后触发搜索，自动刷新列表
  - 数据规则：匹配稿件标题，多关键词空格分隔取交集，模糊匹配

#### 稿件列表

- **列表表头**：复选框（全选）、稿件标题、来源、稿件类型、字数、发布站点、发布时间

- **复选框**
  - 展示规则：每行一个复选框，表头有全选复选框
  - 交互规则：支持单行勾选和表头全选，选中行高亮背景

- **稿件标题**
  - 展示规则：品牌色文字
  - 交互规则：点击进入详情页

- **来源**
  - 展示规则：显示来源机构名称（如"平阳县融媒体中心"）

- **稿件类型**
  - 展示规则：显示稿件类型标签（如"新闻"、"评论"）

- **字数**
  - 展示规则：显示正文字数

- **发布站点**
  - 展示规则：显示发布站点名称（如"爱平阳客户端"）

- **发布时间**
  - 展示规则：格式 YYYY/MM/DD HH:mm:ss

#### 批量取稿

- **取稿按钮**
  - 展示规则：列表上方显示"取稿"按钮，括号内显示已选数量
  - 交互规则：未选中稿件时按钮置灰不可点击；选中后按钮可用，点击弹出取稿弹窗

- **取稿弹窗**
  - 展示规则：弹窗内包含渠道下拉选择和栏目下拉选择
  - 交互规则：
    - 渠道为下拉单选，选中后动态加载该渠道下的栏目列表
    - 栏目为下拉单选，选项根据所选渠道级联更新
    - 底部显示"确认取稿"和"取消"按钮
  - 数据规则：
    - **取数规则**：渠道列表从系统渠道配置读取；栏目列表根据所选渠道动态加载
    - **数据筛选规则**：单选
    - **数据排序规则**：按配置顺序

- **确认取稿**
  - 交互规则：确认后关闭弹窗，列表刷新，显示取稿成功提示

#### 分页

- **展示规则**：页面底部固定分页栏，格式为：共 N 条 | 每页条数下拉 | 上一页 | 页码按钮 | 下一页 | 前往输入框
- **交互规则**：
  - 点击页码跳转对应页，当前页高亮
  - 上一页/下一页按钮，首尾页时禁用
  - "前往"输入页码后回车跳转
  - 每页条数下拉切换后刷新列表
- **数据规则**：
  - **取数规则**：按当前页码和每页条数截取数据
  - **数据筛选规则**：无额外筛选
  - **数据排序规则**：保持当前排序

### 3.5 成品图片网格页

**功能描述**：展示全省范围内已共享的成品图片，以卡片网格视图呈现。

**展示规则**：
- 每行展示多张图片卡片，自适应布局
- 每张卡片包含：缩略图、图片标题、发布时间
- 卡片左上角有复选框，支持批量选择

**交互规则**：
- 点击卡片进入图片详情
- 支持左侧地域筛选和顶部渠道 Tab
- 复选框支持单独勾选

**数据规则**：
- **取数规则**：读取成品图片数据，包含标题、缩略图、来源、渠道、发布时间
- **数据筛选规则**：渠道 Tab 单选 + 地域多选取并集，条件间取交集
- **数据排序规则**：默认按发布时间倒序

**分页**：与成品稿件列表页相同的分页交互

### 3.6 成品视频网格页

**功能描述**：展示全省范围内已共享的成品视频，以卡片网格视图呈现。

**展示规则**：
- 每行展示多个视频卡片，自适应布局
- 每张卡片包含：视频封面缩略图、播放按钮图标（居中叠加于封面）、时长标签（右下角叠加于封面）、视频标题、发布时间
- 播放按钮为半透明白色圆形，内含三角形播放图标
- 时长标签为半透明黑色背景加白色文字，格式为分:秒
- 卡片左上角有复选框，支持批量选择

**交互规则**：
- 点击卡片进入视频详情或播放
- 支持左侧地域筛选和顶部渠道 Tab
- 复选框支持单独勾选

**数据规则**：
- **取数规则**：读取成品视频数据，包含标题、封面图、来源、渠道、发布时间、时长
- **数据筛选规则**：渠道 Tab 单选 + 地域多选取并集，条件间取交集
- **数据排序规则**：默认按发布时间倒序

**分页**：与成品稿件列表页相同的分页交互

### 3.7 稿件详情页

**功能描述**：展示单篇稿件的完整内容。

**展示规则**：
- 左侧主区域：稿件标题 + 元信息（发布站点、渠道、发布时间）+ 正文内容
- 右侧辅助区：结构化信息面板（未分析时显示"暂无结构化信息"和"去分析"按钮）

**交互规则**：
- 左上角返回按钮，点击返回列表页并保留筛选状态
- 进入详情页时左侧地域边栏自动隐藏
- 从详情页返回时恢复边栏和之前的筛选状态

## 4. 交互规范

- 所有筛选条件变更后自动刷新列表
- 无数据时显示"暂无数据"空状态提示
- 从列表页进入详情页时左侧地域边栏自动隐藏
- 从详情页返回时恢复边栏和之前的筛选状态
- 三个子页面共享顶部导航、渠道 Tab、左侧地域筛选和分页交互
- 右上角提供 PRD 切换按钮（绿色圆点 + "PRD"文字）和更新记录按钮
- PRD 面板支持拖拽调整宽度
