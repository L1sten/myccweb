## 1. 产品概述

### 1.1 背景

现有资源中心仅包含"网络资源库"，提供网络稿件、视频、图片、音频等素材类资源的检索与浏览。全省各级媒体单位产出的成品稿件（新闻、专题报道等）缺乏统一的汇聚入口，市县融媒体中心的优质内容难以被省级媒体发现和选用。

### 1.2 产品定位

面向全省媒体从业人员的成品内容汇聚平台，汇聚省本级、浙报集团及各市县融媒体中心已发布的成品类内容（稿件、图片、视频），支持按渠道、地域、时间等多维度检索和浏览，促进优质内容在全省范围内的流通与复用。

### 1.3 用户角色

- **省级媒体编辑**：浏览、筛选市县优质内容，择优选用或二次加工
- **市县融媒体中心编辑**：查看其他地区内容，了解全省报道动态
- **平台管理员**：管理发布规则、审核内容

### 1.4 核心功能概览

在资源中心导航下拉菜单中新增"全省资源库"入口（与"网络资源库"平级），提供三个子模块：成品稿件（列表视图）、成品图片（卡片网格视图）、成品视频（卡片网格视图），支持多条件筛选、地域侧栏筛选、批量取稿和分页功能。

### 1.5 与搜索中心的关系

搜索中心是智能媒资系统的全局检索入口，支持精准检索、文搜图、图搜图三种搜索模式。

**搜索中心对全省资源库的筛选**：

- **展示规则**：搜索中心精准检索模式下，资源库筛选条件包含"全省资源库"单选项，与"全部""网络资源库""个人素材库"并列
- **交互规则**：选中"全省资源库"后，搜索结果仅展示全省资源库中的成品内容（稿件、图片、视频）
- **数据规则**：
  - **取数规则**：从全省资源库读取成品内容数据，包含标题、缩略图、来源、渠道、发布时间等字段
  - **数据筛选规则**：选中"全省资源库"时，仅返回来源为全省资源库的数据，与搜索关键词、发布时间等条件取交集；"全部"不筛选资源库来源
  - **数据排序规则**：支持按时间和相关度排序，默认按时间倒序

## 3. 功能需求

### 3.1 导航入口

**功能描述**：在顶部导航栏"资源中心"下拉菜单中新增"全省资源库"入口，与"网络资源库"平级，位于其右侧。

**展示规则**：
- 下拉菜单分为左右两个并列区域
- 左侧区域：网络资源库，含子菜单项（网络稿件、网络图片、网络视频、网络音频），每项前带图标
- 右侧区域：全省资源库，含子菜单项（成品稿件、成品图片、成品视频），每项前带图标，文字高亮显示
- 两个区域之间以竖线分隔
- 当前页面所属子项无跳转，其余子项点击跳转对应页面

**交互规则**：
- 鼠标悬停"资源中心"时展开下拉菜单，移出后自动收起
- 当前页面为全省资源库任一子页面时，"资源中心"导航文字高亮显示
- 下拉菜单内全省资源库区域整体高亮（标题和文字使用品牌色）

### 3.2 顶部渠道 Tab

**功能描述**：页面顶部提供渠道一级分类 Tab，用于按发布渠道筛选内容。三个子页面共用相同的 Tab 结构。

- **客户端**：展示客户端渠道内容（默认选中）
- **网站**：仅展示发布于网站的内容（本期没有）

**展示规则**：
- Tab 位于页面顶部，横向排列
- 当前选中项文字高亮并显示底部边线

**交互规则**：
- 点击 Tab 切换对应渠道，高亮当前选中
- 切换后列表或网格自动刷新

**数据规则**：
- **取数规则**：匹配账号的"渠道"字段
- **数据筛选规则**：单选，选中后仅显示对应渠道；"全部"不筛选
- **数据排序规则**：保持当前排序设置不变

### 3.3 左侧地域筛选

**功能描述**：左侧边栏按地域分组展示可折叠的地区列表，支持多选和"仅筛选此项"快捷操作。三个子页面共用相同的侧栏交互。

**展示规则**：
- 侧栏采用可折叠分组结构
- 分组节点包括：省本级、11 个地级市（杭州市、湖州市、宁波市、温州市、嘉兴市、绍兴市、金华市、衢州市、舟山市、台州市、丽水市）
- 每个分组节点前有复选框和展开/折叠箭头图标
- 展开后显示子节点列表，每个子节点前有复选框
- 子节点列表：
  - 省本级为独立复选框，无下级子节点
  - 各地级市下：市本级（固定为第一个子项）+ 该市所辖区县
- 子节点悬停时右侧显示漏斗图标
- 漏斗图标悬停时展示提示文字"仅筛选此项"
- 分组节点的展开/折叠按钮旁不显示漏斗图标

**交互规则**：
- 点击分组节点展开/折叠子节点
- 勾选分组节点时，该节点下所有子节点全部选中
- 勾选子节点时，仅选中或取消该节点，同组其他已选项保持不变
- 点击漏斗图标触发"仅筛选此项"：漏斗图标为开关式交互，首次点击变为实心漏斗（选中态），列表仅展示该子项对应地域的数据；再次点击变为空心漏斗（非选中态），恢复为复选框合并结果。漏斗筛选不改变已有复选框状态，当用户手动勾选或取消任何复选框时自动解除漏斗选中态，恢复为复选框合并结果
- 选中后自动刷新列表

**数据规则**：
- **取数规则**：根据内容的"发布站点”在后台管理模块里的地域和级别字段，如"萧山发布客户端"→"萧山区”“区县级”开吧客户端”→“杭州市”“市本级"，省本级的地域为浙江省、省本级。
- **数据筛选规则**：按区县级地域精确匹配，勾选子节点时仅匹配该区县，勾选地级市分组节点时匹配该市下所有子项（含市本级），多选取并集（OR 逻辑），与其他筛选条件取交集
- **数据排序规则**：不影响排序

### 3.4 成品稿件列表页

**功能描述**：展示全省范围内的成品稿件列表，支持多条件筛选、排序、批量取稿和分页。

#### 筛选条件栏

- **日期范围**
  - 展示规则：两个日期输入框
  - 交互规则：选择日期后自动刷新列表
  - 数据规则：默认最近一个月；匹配稿件的"发布时间"字段，取日期范围区间

- **稿件类型**
  - 展示规则：下拉选择
  - 交互规则：选中后自动刷新列表
  - 数据规则：选项为全部稿件、新闻、评论、专题、图集、视频稿、链接稿；匹配稿件的"稿件类型"字段，精确匹配

- **排序**
  - 展示规则：下拉选择
  - 交互规则：选中后自动刷新列表
  - 数据规则：选项为时间倒序、字数正序、字数倒序；默认按时间倒序

- **搜索**
  - 展示规则：搜索输入框
  - 交互规则：输入关键词后触发搜索，自动刷新列表
  - 数据规则：匹配稿件标题，多关键词空格分隔取交集，模糊匹配

#### 稿件列表

- **列表表头**：复选框（全选）、稿件标题、来源、稿件类型、字数、发布站点、发布时间、操作

- **复选框**
  - 展示规则：每行一个复选框，表头有全选复选框
  - 交互规则：支持单行勾选和表头全选，选中行高亮背景

- **稿件标题**
  - 展示规则：品牌色文字
  - 交互规则：点击进入详情页

- **来源**
  - 展示规则：显示来源机构名称（如"平阳县融媒体中心"）

- **稿件类型**
  - 展示规则：显示稿件类型标签（如"新闻"、"评论"）

- **字数**
  - 展示规则：显示正文字数

- **发布站点**
  - 展示规则：显示发布站点名称（如"爱平阳客户端"）

- **发布时间**
  - 展示规则：格式 YYYY/MM/DD HH:mm:ss

- **操作**
  - 展示规则：每行显示"取稿"按钮
  - 交互规则：点击弹出取稿弹窗（与批量取稿弹窗相同）

#### 批量取稿

- **取稿按钮**
  - 展示规则：列表上方显示"取稿"按钮，括号内显示已选数量
  - 交互规则：未选中稿件时按钮置灰不可点击；选中后按钮可用，点击弹出取稿弹窗

- **取稿弹窗**
  - 展示规则：
    - 弹窗标题为"选择取稿渠道"，右上角关闭按钮
    - 弹窗主体为四列表格，列标题为：渠道、站点、频道及稿库、选择取至草稿箱
    - 渠道列：树形展开，显示一级渠道节点（客户端、网站、个人创作中心），每项前带展开/折叠箭头
    - 站点列：展开渠道后显示该渠道下的站点列表，每项前带展开/折叠箭头
    - 频道及稿库列：展开站点后显示该站点下的频道/稿库列表，点击选中高亮
    - 选择取至草稿箱列：显示"取至个人草稿箱"和"取至公共草稿箱"两个复选框
    - 底部左侧显示已选渠道信息，右侧显示"取消"和"确定"按钮
  - 交互规则：
    - 点击渠道行展开/折叠该渠道下的站点列表
    - 点击站点行展开/折叠该站点下的频道及稿库列表
    - 点击频道及稿库项选中该目标，选中项高亮显示，切换选中时自动更新底部已选渠道信息
    - 草稿箱复选框支持勾选，可多选
    - 点击"确定"校验是否已选择渠道和稿库，未选则提示
  - 数据规则：
    - **取数规则**：渠道、站点、频道及稿库三级数据由天目蓝云提供相关接口获取
    - **数据筛选规则**：渠道和稿库单选，草稿箱复选框可多选
    - **数据排序规则**：按接口返回顺序

- **确认取稿**
  - 交互规则：确认后关闭弹窗，列表刷新，显示取稿成功提示

#### 分页

- **展示规则**：页面底部固定分页栏，格式为：共 N 条 | 每页条数下拉 | 上一页 | 页码按钮 | 下一页 | 前往输入框
- **交互规则**：
  - 点击页码跳转对应页，当前页高亮
  - 上一页/下一页按钮，首尾页时禁用
  - "前往"输入页码后回车跳转
  - 每页条数下拉切换后刷新列表
- **数据规则**：
  - **取数规则**：按当前页码和每页条数截取数据
  - **数据筛选规则**：无额外筛选
  - **数据排序规则**：保持当前排序

**展示规则**：

- 每张卡片包含：缩略图、图片标题、发布时间

**交互规则**：
- 点击卡片进入图片详情
- 支持左侧地域筛选和顶部渠道 Tab
- 复选框支持单独勾选

**数据规则**：
- **取数规则**：读取成品图片数据，包含标题、缩略图、来源、渠道、发布时间
- **数据筛选规则**：渠道 Tab 单选 + 地域多选取并集，条件间取交集
- **数据排序规则**：默认按发布时间倒序

**分页**：与成品稿件列表页相同的分页交互

### 3.6 成品视频网格页

**功能描述**：展示全省范围内的成品视频，以卡片网格视图呈现。

**展示规则**：
- 每行展示多个视频卡片，自适应布局
- 每张卡片包含：视频封面缩略图、播放按钮图标（居中叠加于封面）、时长标签（右下角叠加于封面）、视频标题、发布时间

- 时长标签为半透明黑色背景加白色文字，格式为分:秒
- 卡片左上角有复选框，支持批量选择

**交互规则**：
- 点击卡片进入视频详情或播放
- 支持左侧地域筛选和顶部渠道 Tab
- 复选框支持单独勾选

**数据规则**：
- **取数规则**：读取成品视频数据，包含标题、封面图、来源、渠道、发布时间、时长
- **数据筛选规则**：渠道 Tab 单选 + 地域多选取并集，条件间取交集
- **数据排序规则**：默认按发布时间倒序

**分页**：与成品稿件列表页相同的分页交互

### 3.7 成品音频列表页

**功能描述**：展示全省范围内的成品音频，以表格列表视图呈现。

**展示规则**：

**交互规则**：

- 支持左侧地域筛选和顶部渠道 Tab
- 复选框支持单独勾选和全选

**数据规则**：
- **取数规则**：读取成品音频数据，包含标题、时长、文件大小、格式、来源、渠道、发布时间
- **数据筛选规则**：渠道 Tab 单选 + 地域多选取并集，条件间取交集
- **数据排序规则**：默认按发布时间倒序

**分页**：与成品稿件列表页相同的分页交互

### 3.8 音频详情页

**功能描述**：展示成品音频的播放界面和详细信息。

**展示规则**：
- 全屏遮罩层，不影响列表页状态
- 顶部工具栏：返回按钮（左侧）+ 操作按钮居中展示（查看、下载、取至个人）
- 左侧主区域：
  - 音频播放器（音符图标 + 播放按钮 + 进度条 + 时间显示 + 音量图标）
  - 信息面板分为两列：基本信息（标题、发布站点、渠道、上传时间）+ 音频数据（时长、大小、格式）
- 右侧信息面板：基本信息 Tab + 关键词区域

**交互规则**：
- 点击音频标题进入详情页
- 点击返回按钮关闭详情页

**数据规则**：
- **取数规则**：读取音频数据，包含标题、来源、渠道、上传时间、时长、文件大小、格式、关键词
- **数据筛选规则**：无额外筛选
- **数据排序规则**：无排序

### 3.9 稿件详情页

**功能描述**：从成品稿件列表点击稿件标题进入稿件详情页，并根据稿件类型展示对应详情模板。

**展示规则**：
- 全屏遮罩层，不影响列表页状态
- 顶部工具栏：返回按钮（左侧）+ 操作按钮居中展示（取稿、阅读原文）
- 左侧主区域统一包含稿件标题、稿件基础元信息和类型化正文内容
- 新闻稿详情：展示新闻正文，可包含正文图片
- 图集稿详情：展示正文说明 + 图集轮播区，轮播区包含主图、当前序号、左右切换按钮和缩略图列表
- 视频稿详情：展示视频播放区 + 正文说明，视频播放区包含封面、播放按钮、进度条、时间和基础播放控件
- 链接稿详情：展示外链地址，点击后打开原始页面
- 右侧信息面板展示结构化信息，包含稿件信息、关键词和摘要
- 图集稿、视频稿、链接稿、新闻稿的结构化信息字段保持一致

**交互规则**：
- 点击稿件标题时，根据该行稿件类型进入对应详情模板
- 点击返回按钮关闭详情页

## 4. 权限点一览

全省资源库的权限点按资源类型分组，每个权限点为独立开关，支持按角色配置。

| 资源类型 | 权限点 |
| 成品稿件 | 查看、取稿、阅读原文、结构化分析 |
| 成品图片 | 查看、下载、添加到个人素材库 |
| 成品视频 | 查看、下载、添加到个人素材库 |
| 成品音频 | 查看、下载、添加到个人素材库 |

**展示规则**：
- 权限配置以树形表格展示，左列为资源类型及权限点，右列为勾选状态
- 资源类型为一级节点，权限点为二级节点，支持展开/折叠
- 各资源类型默认展开

**交互规则**：
- 勾选资源类型时，该类型下所有权限点全部选中
- 取消勾选资源类型时，该类型下所有权限点全部取消
- 勾选/取消单个权限点时，不影响同类型其他权限点
- 所有权限点均支持独立开关

**数据规则**：
- **取数规则**：从权限配置模块读取角色对应的权限点列表
- **数据筛选规则**：按资源类型分组展示，每个权限点为布尔值（启用/禁用）
- **数据排序规则**：按资源类型固定顺序（成品稿件→成品图片→成品视频→成品音频），权限点按列表顺序
